In HTML, elements are categorized into **inline** and **block-level** elements based on how they behave in the document flow.

### ğŸ§± Block-Level Elements

A **block-level element** always starts on a **new line** and takes up the **full width** available (by default). It essentially forms a block or a box on the page.

**Characteristics**

- Begins on a new line.
- Expands to fill the container's width by default.
- Can contain **other block** and **inline** elements.
- Typically used for **structural layout**.

### ğŸ”¡ Inline Elements

An **inline element** does **not start on a new line** and only takes up as much width as necessary. It flows alongside other elements like words in a sentence.

**Characteristics**

* Does **not break** the line.
* Only takes up as much width as required.
* Cannot contain **block-level** elements (only text or other inline elements).
* Commonly used for **text formatting**.

### âœ… Common Inline Elements

```html
<span></span>
<a></a>
<strong></strong>
<b></b>, <i></i>
<img />
<code></code>
```

**Example:**

    <a class="inline">I am a </a>
    <b class="inline">I am b </b>

  </body>
</html> 
`}/>
