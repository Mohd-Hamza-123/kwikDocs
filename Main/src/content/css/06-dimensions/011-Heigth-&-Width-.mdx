The CSS `height` and `width` properties are used to set the height and width of an element.


**Example:**

```css
.container{
  height: 100px; /* Sets the height of the element to 100 pixels */
  width: 500px; /* Sets the width of the element to 500 pixels */
  background-color: powderblue; /* Sets a light blue background color */
}
```

**Example:**
<HtmlEditor id={1} defaultCode={`<html>
  <head>
    <style>
      .container{
        height: 100px;
        width: 500px;
        border: 2px solid black;
      }
    </style>
  </head>
  <body>
    <div class="container">
      <p>This is a paragraph with a height of 100 pixels and a width of 500 pixels.</p>
    </div>
  </body>
</html>
`}/>


**Important Point**

* When we set the height and width of `html` element, it will cover the whole screen. Height 100% means whole screen height and width 100% means whole screen width.
