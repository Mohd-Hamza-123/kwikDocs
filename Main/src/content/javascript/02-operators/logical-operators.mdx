### ğŸ”¹ **1. `&&` (Logical AND)**

#### **Definition:**

Returns `true` **only if both** operands are true. If the first value is falsy, it returns that; otherwise, it returns the second value.

#### **Example:**

```js
let isVerified = true;
let isAdmin = true;

let canAccess = isVerified && isAdmin;

console.log(canAccess); // true
```

ğŸ“ **Explanation:** Both `isVerified` and `isAdmin` are `true`, so `canAccess` becomes `true`.

### ğŸ”¹ **2. `||` (Logical OR)**

#### **Definition:**

Returns `true` if **at least one** operand is true. It returns the first truthy value it finds, or the last falsy one if none are true.

#### **Example:**

```js
let isVerified = false;
let isAdmin = true;

let canAccess = isVerified || isAdmin;

console.log(canAccess); // true
```

ğŸ“ **Explanation:** One of the conditions is `true`, so `canAccess` becomes `true`.

---

### ğŸ”¹ **3. `!` (Logical NOT)**

#### **Definition:**

Inverts the truthiness of a value. Turns `true` into `false` and `false` into `true`.

#### **Example:**

```js
let isBanned = false;

let canLogin = !isBanned;

console.log(canLogin); // true
```

ğŸ“ **Explanation:** `isBanned` is `false`, so `!isBanned` becomes `true`. The user can log in.
