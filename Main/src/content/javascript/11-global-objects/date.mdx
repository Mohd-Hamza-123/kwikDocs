## ğŸ•’ Formatting Dates

```js
const d = new Date();
console.log(d.toString());          // Sun Aug 03 2025 14:37:16 GMT+0530 (India Standard Time)
console.log(d.toDateString());      // Sun Aug 03 2025
console.log(d.toTimeString());      // 14:37:16 GMT+0530 (India Standard Time)
console.log(d.toISOString());       // 2025-08-03T09:07:16.557Z
console.log(d.toLocaleDateString()); // 3/8/2025
console.log(d.toLocaleTimeString()); // 2:37:16 pm
```

## ğŸ” Compare Dates

You can compare dates by converting them to **timestamps**.

```js
const date1 = new Date('2025-08-01');
const date2 = new Date('2025-08-03');

console.log(date1.getTime() === date2.getTime()); // false
console.log(date1.getTime() < date2.getTime());   // true
console.log(+date1 === +date2);                   // also valid
```

---

>  Use `getTime()` or `+date` to get numeric values for accurate comparisons.
