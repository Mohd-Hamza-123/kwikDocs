2. MySQL Database : 

```python
# settings.py
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',  # Database backend
        'NAME': 'your_database_name',          # Name of your MySQL database
        'USER': 'your_database_user',          # MySQL username
        'PASSWORD': 'your_database_password',  # MySQL password
        'HOST': 'localhost',                   # Database host (default is localhost)
        'PORT': '3306',                        # Database port (default is 3306)
        'OPTIONS': {
            'init_command': "SET sql_mode='STRICT_TRANS_TABLES'",  # Recommended SQL mode
        },
    }
}
```
