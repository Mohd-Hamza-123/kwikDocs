1. Postgresql Database : 

If you don't want to use `SQLite`, you can use `PostgreSQL` or `MySQL`. Here is the database settings to set up `PostgreSQL` or `MySQL` in Django.

```python
# settings.py
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'your_db_name',
        'USER': 'your_db_user',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```
