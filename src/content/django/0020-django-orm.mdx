Django ORM (Object-Relational Mapping) is a feature of the Django web framework that allows developers to interact with a relational database(sql lite,mySQL,Postgresql, etc) using Python code instead of writing raw SQL queries.


<ImageCenter src="https://res.cloudinary.com/dkkifg17t/image/upload/v1744537134/Documentarium/Django/nywm62hoivevkzg0aqfd.png" alt="Django ORM"/> 


### Key Features of Django ORM:

- Tables in the database are represented as Python classes.
- Each table column corresponds to a class attribute.
- Each row in the table corresponds to an instance of the class.

### Example :

```python
# models.py
from django.db import models
class Book(models.Model):
    title = models.CharField(max_length=100)
    author = models.CharField(max_length=50)
    published_date = models.DateField()
    price = models.DecimalField(max_digits=6, decimal_places=2)
```
