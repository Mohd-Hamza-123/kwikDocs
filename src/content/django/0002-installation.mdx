### Check if Django is already installed 

```bash
django-admin --version
```

### Installing virtualenv

1. First Install Virtual env 
```bash
pip install virtualenv
```

2. Creating Virtual Env

```bash
virtualenv "Env Name"
virtualenv venv
# or
python -m virtualenv "Env Name" 
python -m virtualenv venv 
```

3. Activating Virtual env 

```bash
# For Windows 
"Env Name"\Scripts\activate
```

```bash
# For Mac & Linux
source "Env Name"/bin/activate
```

4. Deactivating Virtual env 

```bash
deactivate
```

### Installing Django 

```bash
pip install django
# want to download a specific version
pip install django=5.0
```
