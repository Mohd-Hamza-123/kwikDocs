Function-based views (FBVs) are simple Python functions that take a request and return a response.

#### Creating a Simple Function Based View

```python
# views.py
from django.http import HttpResponse

def home_view(request):
    return HttpResponse("Hello, world!. This is home Page")

def about_view(request):
    return HttpResponse("This is about Page")

def contact_view(request):
    return HttpResponse("<h1><u>This is contact Page</u></h1>")
```

URL Configuration for above `Views`

```python
# urls.py
from django.urls import path
from appName.views import home_view,about_view

urlpatterns = [
    path("",home_view),
    path("about/",about_view),
    path("contact/",contact_view)
]
```
