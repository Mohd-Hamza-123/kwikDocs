In **React Router DOM**, both **`Link`** and **`NavLink`** are used for navigation, but they have **different purposes**:


### Link

* Used to navigate between routes.
* Provides simple navigation without page reloads.
* Does **not** know if the link is currently active.

Example:

```jsx
import { Link } from "react-router-dom";

<Link to="/about">About</Link>
```

This just takes you to `/about`. Nothing extra.


### NavLink

* Works the same as `Link` but with **extra styling behavior**.
* It **knows if the route is active** and allows you to style the link differently when active.
* Very useful for **navigation menus, sidebars, or tabs**.

Example:

```jsx
import { NavLink } from "react-router-dom";

<NavLink 
  to="/about" 
  className={({ isActive }) => (isActive ? "text-blue-500 font-bold" : "text-gray-500")}
>
  About
</NavLink>
```

**Example: Navbar with `NavLink`**

```tsx
"use client";
import { BrowserRouter as Router, Routes, Route, NavLink } from "react-router-dom";

function Navbar() {
  const navItems = [
    { path: "/", label: "Home" },
    { path: "/about", label: "About" },
    { path: "/services", label: "Services" },
    { path: "/contact", label: "Contact" },
  ];

  return (
    <nav>
      <ul>
        {navItems.map((item) => (
          <li key={item.path}>
            <NavLink
              to={item.path}
              end // ensures exact matching for root "/"
              className={({ isActive }) =>
                `px-3 py-2 rounded-md text-sm font-medium transition ${
                  isActive
                    ? "bg-blue-600 text-white"
                    : "text-gray-300 hover:bg-gray-700 hover:text-white"
                }`
              }
            >
              {item.label}
            </NavLink>
          </li>
        ))}
      </ul>
    </nav>
  );
}
```

**How it works**

* When you click a link, it **navigates without refreshing**.
* The **active link** gets a blue background (`bg-blue-600 text-white`).
* Inactive links stay gray but change on hover.

* Use `Link` when you just need navigation.
* Use `NavLink` when you need to highlight which route is currently active.
