title: Not Found Page
published: true
---

In **React Router v7**, you can handle unknown routes by using a **catch-all route** (`*`).
This is typically used to show a **404 Not Found** page.

**Basic Example (using `Component`)**

```tsx
import { Routes, Route } from "react-router-dom";

function App() {
  return (
    <Routes>
      <Route path="/" Component={Home} />
      <Route path="/about" Component={About} />

      {/* Catch-all route */}
      <Route path="*" Component={NotFound} />
    </Routes>
  );
}

function NotFound() {
  return <h1>404 - Page Not Found</h1>;
}
```


**Example with `element` (when passing props)**

If your **Not Found page** needs custom props, use `element` instead:

```tsx
<Route path="*" element={<NotFound customMessage="Oops! Page not found" />} />
```

### Alternative: Redirect Unknown Routes

If you prefer to redirect users instead of showing a 404 page, you can use `<Navigate />`

```jsx
import { Navigate } from "react-router-dom";

<Route path="*" element={<Navigate to="/" replace />} />
```
