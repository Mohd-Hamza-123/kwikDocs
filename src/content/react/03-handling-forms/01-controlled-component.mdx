In **React**, a **controlled component** is a form element (like `<input>`, `<textarea>`, or `<select>`) whose value is **controlled by React state** instead of the DOM.

That means:

* The **source of truth** is React state (`useState`, Redux, etc.).
* The form element only **displays** the value from state.
* When the user types or changes something, React updates the state, and then the state updates the input.

```tsx
import { useState } from "react";

function ControlledInput() {
  const [name, setName] = useState(""); // React state controls input value

  return (
    <div>
      <input
        type="text"
        value={name}                 // value comes from state
        onChange={(e) => setName(e.target.value)} // updates state on change
      />
      <p>Hello, {name}!</p>
    </div>
  );
}

export default ControlledInput;
```
