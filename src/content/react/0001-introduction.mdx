### Installation 

To install React, you need to have Node.js installed on your machine. You can download Node.js from the official website. Once you have Node.js installed, you can install React using the following command:

```bash
npm create vite@latest
```

To run the development server, you can use the following command:

```bash
npm run dev
```

run `npm install vite` if it shows 'vite' is not recognized as an internal or external command,
