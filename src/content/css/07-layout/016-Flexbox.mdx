Flexbox is short for the Flexible Box Layout module. It is a layout method for arranging items in rows or columns.
An Element become flexbox by setting the `display` property to `flex`.

### ðŸŒŸ CSS Flexbox Components

A flexbox always consists of:

- **Flex Container**: The parent (container).
- **Flex Items**: The items inside the container.


In below example , the div with class `flex-container` is the flex container(parent) and the div with class `flex-items` are the flex items(child).

<HtmlEditor id={0} defaultCode={`<html>
<head>
  <style>
    .flex-container {
      display: flex;
    }
    .flex-items {
    height: 50px;
    width: 50px;
    background-color: lightblue;
    border: 2px solid black;
    }
  </style>
</head>
<body>
  <div class="flex-container">
    <div class="flex-items">1</div>
    <div class="flex-items">2</div>
    <div class="flex-items">3</div>
  </div>
</body>
</html>
`}/>


When the Element become flexbox, all its children directions are set to `row` by default.


### ðŸš€ CSS Properties for Flex Container

- **`flex-direction`**
- **`flex-wrap`**
- **`justify-content`**
- **`align-items`**
- **`align-content`**


#### **1. Flex Direction**

This property specifies the display direction of flex items in the flex container.

```css
.flex-container {
  display: flex;
  flex-direction: row;
}
```

#### **2. Flex Wrap**

The `flex-wrap` property specifies whether the flex items should wrap or not.

```css
.flex-container {
  display: flex;
  flex-wrap: nowrap;
}
```

<ImageCenter src="https://res.cloudinary.com/dkkifg17t/image/upload/v1737019967/Documentarium/JavaScript/gsbcxnfqnx4uweepmlqs.png" alt="Flex Wrap" />


<HtmlEditor id={2} defaultCode={`<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Flex Wrap Example</title>
  <style>
    body {
     padding:5px;
    }

    section {
      border: 2px solid red;
      display: flex;
      flex-direction: row; /* default is row */
      /* height: 200px; */
      width: 350px;
    }

    section div {
      border: 2px solid black;
      padding: 5px;
      width: 100px; /* fixed width so wrapping is visible */
      text-align: center;
    }

    /* Flex-wrap variations */
    .nowrap {
      flex-wrap: nowrap;
    }

    .wrap {
      flex-wrap: wrap;
    }

    .wrap-reverse {
      flex-wrap: wrap-reverse;
    }
  </style>
</head>
<body>

  <div>
    <h2>No Wrap (default)</h2>
    <section class="nowrap">
      <div>1</div><div>2</div><div>3</div><div>4</div>
      <div>5</div><div>6</div><div>7</div><div>8</div>
    </section>
  </div>

  <div>
    <h2>Wrap</h2>
    <section class="wrap">
      <div>1</div><div>2</div><div>3</div><div>4</div>
      <div>5</div><div>6</div><div>7</div><div>8</div>
    </section>
  </div>

  <div>
    <h2>Wrap Reverse</h2>
    <section class="wrap-reverse">
      <div>1</div><div>2</div><div>3</div><div>4</div>
      <div>5</div><div>6</div><div>7</div><div>8</div>
    </section>
  </div>

</body>
</html>
`}/>

#### **3. Justify Content**

The `justify-content` property is used to align the flex items when they do not use all available space on the main axis (horizontally).

```css
.flex-container {
  display: flex;
  justify-content: center;
}
```

#### **4. Align Items**

The `align-items` property is used to align the flex items when they do not use all available space on the cross axis (vertically).

```css
.flex-container {
  display: flex;
  height: 200px;
  align-items: center;
}
```

### **6. Gap**

The `gap` property in flexbox is used to set the gaps between the flex items, which is the space between the rows or columns.

```css
.flex-container {
  display: flex;
  gap: 10px;
}
```
