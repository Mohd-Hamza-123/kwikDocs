The CSS `margin` and `padding` properties are used to create space around and within elements, respectively.

---

### üé® CSS Example:

```css
.container {
  margin: 20px; /* Adds space outside the element */
  padding: 15px; /* Adds space inside the element */
  background-color: lightgray; /* Adds a light gray background for visualization */
}

.child {
  margin: 10px 5px; /* Adds 10px vertically and 5px horizontally outside the element */
  padding: 8px 12px; /* Adds 8px vertically and 12px horizontally inside the element */
  background-color: white; /* Adds a white background for visualization */
}
```
