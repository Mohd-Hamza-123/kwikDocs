### Padding 

The `padding` property is used to add space inside the border of an element. It is a shorthand property for `padding-top`, `padding-right`, `padding-bottom`, and `padding-left`.

**Example:**

```css
.container {
  padding: 10px; /* Adds 10px of padding to all sides */
}

.container {
  padding: 10px 20px; /* Adds 10px of padding to the top and bottom, and 20px of padding to the left and right */
}

.container {
  padding: 10px 20px 30px; /* Adds 10px of padding to the top, 20px of padding to the right, and 30px of padding to the bottom */
}

.container {
  padding: 10px 20px 30px 40px; /* Adds 10px of padding to the top, 20px of padding to the right, 30px of padding to the bottom, and 40px of padding to the left */
}
.container {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
} 
```

### Margin

The `margin` property is used to add space outside the border of an element. It is a shorthand property for `margin-top`, `margin-right`, `margin-bottom`, and `margin-left`.

**Example:**

```css
.container {
  margin: 10px; /* Adds 10px of margin to all sides */
}

.container {
  margin: 10px 20px; /* Adds 10px of margin to the top and bottom, and 20px of margin to the left and right */
}

.container {
  margin: 10px 20px 30px; /* Adds 10px of margin to the top, 20px of margin to the right, and 30px of margin to the bottom */
}

.container {
  margin: 10px 20px 30px 40px; /* Adds 10px of margin to the top, 20px of margin to the right, 30px of margin to the bottom, and 40px of margin to the left */
} 

.container {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
}
```
