title: CSS Font Family
published: true
---

In CSS, we use the `font-family` property to specify the font of a text.


**Example:**

```css
.p1 {
  font-family: "Times New Roman", Times, serif; /* Specifies a serif font */
}

.p2 {
  font-family: Arial, Helvetica, sans-serif; /* Specifies a sans-serif font */
}

.p3 {
  font-family: "Lucida Console", "Courier New", monospace; /* Specifies a monospace font */
}
```
