You can target **screen width** in CSS media queries using **`min-width`**, **`max-width`**, or both.


### ðŸ”¹ 1. Change styles for screens **up to** a certain width

ðŸ‘‰ Use **`max-width`** to apply styles when the screen is **narrower** than the value.

```css
/* Styles for screens 480px wide or smaller (like mobile) */
@media (max-width: 480px) {
  body {
    background-color: lightblue;
    font-size: 14px;
  }
}
```

### ðŸ”¹ 2. Change styles for screens **from** a certain width

ðŸ‘‰ Use **`min-width`** to apply styles when the screen is **wider** than the value.

```css
/* Styles for screens 768px and wider (like tablets & desktops) */
@media (min-width: 768px) {
  body {
    background-color: lightgreen;
    font-size: 18px;
  }
}
```


### ðŸ”¹ 3. Change styles **between two widths**

ðŸ‘‰ Combine **`min-width`** and **`max-width`**.

```css
/* For screens between 481px and 768px (like tablets) */
@media (min-width: 481px) and (max-width: 768px) {
  body {
    background-color: lightyellow;
    font-size: 16px;
  }
}
```


**Quick Example:**

```css
/* Mobile: up to 480px */
@media (max-width: 480px) {
  h1 {
    font-size: 18px;
  }
}

/* Tablet: 481px to 768px */
@media (min-width: 481px) and (max-width: 768px) {
  h1 {
    font-size: 24px;
  }
}

/* Desktop: 769px and above */
@media (min-width: 769px) {
  h1 {
    font-size: 32px;
  }
}
```
