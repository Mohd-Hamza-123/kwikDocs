The `position` property specifies the type of positioning method used for an element (static, relative, absolute, fixed, or sticky).

#### **1. Position Relative**

The `relative` position allows an element to be positioned relative to its normal position.

```css
.relative-box {
  position: relative;
  top: 20px; /* Move down 20px from its original position */
  left: 30px; /* Move right 30px from its original position */
  width: 150px;
  height: 100px;
  background-color: coral;
}
```

#### **2. Position Absolute**

The `absolute` position allows an element to be positioned relative to its nearest positioned ancestor (not static).

```html
<div class="container">
  Parent Container
  <div class="absolute-box">
    I'm absolute
  </div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 200px;
  border: 2px solid #333;
}

.absolute-box {
  position: absolute;
  top: 50px;
  left: 50px;
  width: 100px;
  height: 50px;
  background-color: tomato;
  color: white;
}
```

Absolute box will be positioned relative to its parent element.

#### **3. Position Fixed**

The `fixed` position removes the element from the normal document flow and positions it relative to the viewport.

```html
<div class="navbar">Navbar</div>
```

```css
.navbar {
  height: 90px;
  background-color: maroon;
  position: fixed;
  width: 100vw;
  top: 0px;
}
```

The element (here, it is the navbar) remains fixed in the viewport.
