**What is CSS Flexbox?**

Flexbox is short for the Flexible Box Layout module. It is a layout method for arranging items in rows or columns.

---

### ðŸŒŸ CSS Flexbox Components

A flexbox always consists of:

- **Flex Container**: The parent (container).
- **Flex Items**: The items inside the container.

```html
<div class="flex-container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
```

The flex container becomes flexible by setting the `display` property to `flex`:

```css
.flex-container {
  display: flex;
}
```

- **`flex-wrap`**

- **`align-content`**

---

#### **1. Flex Direction**

This property specifies the display direction of flex items in the flex container.

```css
.flex-container {
  display: flex;
  flex-direction: row;
}
```

#### **2. Flex Wrap**

The `flex-wrap` property specifies whether the flex items should wrap or not.

```css
.flex-container {
  display: flex;
  flex-wrap: nowrap;
}
```

<ImageCenter src="https://res.cloudinary.com/dkkifg17t/image/upload/v1737019967/Documentarium/JavaScript/gsbcxnfqnx4uweepmlqs.png" alt="Flex Wrap" />

---

#### **3. Justify Content**

The `justify-content` property is used to align the flex items when they do not use all available space on the main axis (horizontally).

```css
.flex-container {
  display: flex;
  justify-content: center;
}
```

<ImageCenter src="https://res.cloudinary.com/dkkifg17t/image/upload/v1737020026/Documentarium/JavaScript/ggw40hzsm9j7s96ctabj.png" alt="Justify Content" />

---

#### **4. Align Items**

The `align-items` property is used to align the flex items when they do not use all available space on the cross axis (vertically).

```css
.flex-container {
  display: flex;
  height: 200px;
  align-items: center;
}
```

#### **5. Align Content**

The `align-content` property is used to align the flex items when they do not use all available space on the cross axis (vertically).


```css showLineNumbers {5}
.flex-container {
   display: flex;
   flex-wrap: wrap;
   flex-direction: row;
   align-content: center;
   justify-content: center;
}
```

<ImageCenter src="https://res.cloudinary.com/dkkifg17t/image/upload/v1752487098/Screenshot_2025-07-14_152552_so86nu.png" alt="Align Content" />

You can use different values for `align-content` property:

- `flex-start`
- `flex-end`
- `space-between`
- `space-around`
