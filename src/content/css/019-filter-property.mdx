CSS filter property is a powerful tool to manipulate the visual appearance of an element. It can be used to apply a variety of visual effects, such as blur, brightness, contrast, grayscale, hue-rotate, invert, opacity, saturate, sepia, and drop-shadow.

> Filter property can be applied to any element in HTML that is rendered visually.

### 1. **`blur()`**

```css
img { filter: blur(5px); }
```

* **Start (min):** `0` â†’ no blur.
* **End (max):** No strict max, but very large values (`blur(500px)`) = extreme blur + performance hit.

---

### 2. **`brightness()`**

```css
img { filter: brightness(150%); }
```

* **Start:** `0` â†’ completely black.
* **Normal:** `1` or `100%` â†’ unchanged.
* **End:** No strict max â†’ higher = brighter (`brightness(1000%)` makes it almost pure white).

---

### 3. **`contrast()`**

```css
img { filter: contrast(200%); }
```

* **Start:** `0` â†’ flat gray (no contrast).
* **Normal:** `1` or `100%`.
* **End:** No strict max â†’ `contrast(500%)` = extreme contrast.

---

### 4. **`grayscale()`**

```css
img { filter: grayscale(100%); }
```

* **Start:** `0` or `0%` â†’ normal color.
* **End:** `1` or `100%` â†’ fully grayscale.
* (Values >1 are **clamped** to `1`).

---

### 5. **`hue-rotate()`**

```css
img { filter: hue-rotate(90deg); }
```

* **Start:** `0deg` â†’ normal.
* **End:** Any angle works, but practically `0degâ€“360deg`.
* (Values wrap around â†’ `hue-rotate(360deg)` = `0deg`).

---

### 6. **`invert()`**

```css
img { filter: invert(100%); }
```

* **Start:** `0` â†’ normal.
* **End:** `1` or `100%` â†’ fully inverted (like a photo negative).
* (Values >1 are clamped).

---

### 7. **`opacity()`**

```css
img { filter: opacity(50%); }
```

* **Start:** `0` â†’ fully transparent.
* **Normal:** `1` or `100%`.
* **End:** `1` is max â†’ values >1 are clamped.

---

### 8. **`saturate()`**

```css
img { filter: saturate(300%); }
```

* **Start:** `0` â†’ grayscale.
* **Normal:** `1` or `100%`.
* **End:** No strict max â†’ higher = oversaturated colors (e.g., `saturate(500%)`).

---

### 9. **`sepia()`**

```css
img { filter: sepia(100%); }
```

* **Start:** `0` â†’ normal.
* **End:** `1` or `100%` â†’ full sepia (brownish effect).
* (Values >1 are clamped).

---

### 10. **`drop-shadow()`**

```css
img { filter: drop-shadow(10px 10px 5px rgba(0,0,0,0.5)); }
```

* **Start:** No strict min (offsets can be `0`, blur radius can be `0`).
* **End:** No strict max â†’ values depend on `offset-x`, `offset-y`, and `blur-radius`.
