### 🖋️ The _`font-size`_ property sets the size of a font.

---

### 🎨 CSS File:

```css
.para {
  font-size: 24px; /* Sets the font size to 24 pixels */
}
```

---

### 📝 HTML File:

```html
<p class="para">
  I am a paragraph.
</p>
```
