The **`overflow`** property in CSS controls what happens when content inside an element is **too big** to fit into the elementâ€™s box (its width/height).

By default, if text or content is larger than the box, it just spills out â€” but with `overflow`, you can control this behavior.


### **Syntax**

```css
selector {
  overflow: visible | hidden | scroll | auto | clip;
}
```

---

### **Values of `overflow`**

1. **`visible`** (default)

   * Content is not clipped and may overflow outside the element's box.
   * Example: text spilling out of a div.

2. **`hidden`**

   * Extra content is clipped, and not visible outside the box.
   * No scrollbar is shown.

3. **`scroll`**

   * Extra content is clipped, **but scrollbars are always visible** (horizontal and vertical).
   * Even if content fits, scrollbars stay.

4. **`auto`**

   * Scrollbars appear **only when needed**.
   * If content fits â†’ no scrollbar.
   * If content overflows â†’ scrollbar appears.

5. **`clip`** (newer)

   * Similar to `hidden`, but it does not allow scrolling at all (content is clipped to the elementâ€™s box).
