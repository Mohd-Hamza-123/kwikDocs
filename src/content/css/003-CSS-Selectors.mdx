CSS selectors are used to "find" (or select) the HTML elements you want to style.

### 2Ô∏è‚É£ **ID Selector** üîë

The **`id` selector** targets a single HTML element with a unique ID.  
**Note:** An ID name cannot start with a number or special symbols (`*`, `$`, `@`, etc.).

#### Example:

```css

  text-align: center;
  color: red;
}
```

```html

```
