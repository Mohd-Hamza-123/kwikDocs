_**🎨 The `color` property specifies the color of text.**_

---

## 🌈 Using the `color` Property

The `color` property is used to define the color of text in an HTML element. You can specify colors using:

1. **🎨 Color names** (e.g., `red`, `blue`, `green`)
2. **🟦 Hexadecimal values** (e.g., `#FF0000` for red)
3. **🔴 RGB values** (e.g., `rgb(255, 0, 0)` for red)
4. **🌀 HSL values** (e.g., `hsl(0, 100%, 50%)` for red)

---

### 🖍️ Example: Using Color Names

#### CSS File:

```css
.text-color {
  color: red; /* Sets the text color to red */
}
