**Font Sizes in Tailwind CSS**

### 1️⃣ **Default font sizes**

Tailwind provides a scale of font sizes from **extra small to 9xl**:

```html
<p class="text-xs">Extra small</p>
<p class="text-sm">Small</p>
<p class="text-base">Base (default)</p>
<p class="text-lg">Large</p>
<p class="text-xl">Extra large</p>
<p class="text-2xl">2xl</p>
<p class="text-3xl">3xl</p>
<p class="text-4xl">4xl</p>
<p class="text-5xl">5xl</p>
<p class="text-6xl">6xl</p>
<p class="text-7xl">7xl</p>
<p class="text-8xl">8xl</p>
<p class="text-9xl">9xl</p>
```

---

### 2️⃣ **Arbitrary font sizes**

If you need a custom size:

```html
<p class="text-[18px]">Custom 18px</p>
<p class="text-[2.5rem]">Custom 2.5rem</p>
<p class="text-[10vw]">Responsive 10vw</p>
```

---

### 3️⃣ **Responsive font sizes**

Make font size change based on screen size:

```html
<p class="text-sm md:text-lg lg:text-2xl">
  Responsive font size
</p>
```

* `sm:` → on small screens
* `md:` → on medium screens
* `lg:` → on large screens

---

### 4️⃣ **Custom sizes in `tailwind.config.js`**

You can extend font sizes:

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      fontSize: {
        'xxs': '0.6rem',
        'xxxl': '2.5rem',
      },
    },
  },
};
```

Usage:

```html
<p class="text-xxs">Extra extra small</p>
<p class="text-xxxl">Extra extra extra large</p>
```


✅ **Summary**

* Use **`text-{size}`** (e.g. `text-lg`, `text-2xl`)
* Use **arbitrary values** (`text-[20px]`, `text-[1.75rem]`)
* Use **responsive variants** (`sm:text-lg md:text-2xl`)
* Extend **custom sizes** in `tailwind.config.js`
