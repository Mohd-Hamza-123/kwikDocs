**Text Styling in Tailwind CSS**

### 1️⃣ **Italic**

```html
<p class="italic">This text is italic</p>
<p class="not-italic">This text is not italic</p> <!-- overrides -->
```

---

### 2️⃣ **Underline / No underline / Decorations**

```html
<p class="underline">This text is underlined</p>
<p class="overline">This text has an overline</p>
<p class="line-through">This text has a strikethrough</p>
<p class="no-underline">This text has no underline</p>
```

👉 You can also control **decoration color and style**:

```html
<p class="underline decoration-red-500">Red underline</p>
<p class="underline decoration-dotted">Dotted underline</p>
<p class="underline decoration-wavy decoration-blue-600">Wavy blue underline</p>
```

---

### 3️⃣ **Bold (Font Weight)**

Tailwind provides `font-{weight}` classes:

```html
<p class="font-bold">Bold text</p>
<p class="font-semibold">Semi-bold text</p>
<p class="font-medium">Medium weight text</p>
<p class="font-normal">Normal text</p>
<p class="font-light">Light text</p>
```

---

✅ **Summary**

* **Italic** → `italic` / `not-italic`
* **Underline/Decoration** → `underline`, `overline`, `line-through`, `no-underline` (+ `decoration-*` for color/style)
* **Bold** → `font-bold` (or other weights like `font-light`, `font-extrabold`)
