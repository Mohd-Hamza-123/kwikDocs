**Ways to Set Font Color in Tailwind CSS**

### 1Ô∏è‚É£ **Default color palette**

Tailwind comes with a predefined color system.

```html
<p class="text-red-500">Red 500</p>
<p class="text-green-700">Green 700</p>
<p class="text-blue-900">Blue 900</p>
```

üìå Format: `text-{color}-{shade}` (`50` ‚Üí lightest, `900` ‚Üí darkest)

---

### 2Ô∏è‚É£ **Custom colors (extend theme)**

In `tailwind.config.js`, extend your own brand colors:

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        brand: '#FF5733',
      },
    },
  },
};
```

Usage:

```html
<p class="text-brand">Custom brand color</p>
```

---

### 3Ô∏è‚É£ **Arbitrary values** (no config needed)

Use **raw color values** directly:

```html
<p class="text-[#1e40af]">Custom HEX</p>
<p class="text-[rgb(34,197,94)]">Custom RGB</p>
<p class="text-[hsl(200,100%,50%)]">Custom HSL</p>
```

---

### 4Ô∏è‚É£ **State-based colors**

Change color on hover, focus, active, etc.:

```html
<p class="text-gray-700 hover:text-black">Hover me</p>
<p class="text-blue-600 focus:text-blue-800">Focus me</p>
```

---

### 5Ô∏è‚É£ **Dark mode colors**

Apply different colors in dark mode:

```html

```

---

### 6Ô∏è‚É£ **Responsive font colors**

Different font colors on screen sizes:

```html
<p class="text-gray-500 md:text-blue-600 lg:text-green-700">
  Responsive font color
</p>
```


‚úÖ **Summary of Tailwind-only methods**

* **`text-{color}-{shade}`** ‚Üí built-in palette
* **Custom theme colors** ‚Üí via `tailwind.config.js`
* **Arbitrary values** ‚Üí `text-[#hex]`, `text-[rgb()]`, `text-[hsl()]`
* **State variants** ‚Üí `hover:`, `focus:`, `active:`
* **Dark mode** ‚Üí `dark:`
* **Responsive** ‚Üí `sm:`, `md:`, `lg:`, `xl:`
