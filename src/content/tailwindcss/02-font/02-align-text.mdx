**Text Alignment in Tailwind CSS**

### 1️⃣ **Basic alignment**

```html
<p class="text-left">Left aligned text</p>
<p class="text-center">Center aligned text</p>
<p class="text-right">Right aligned text</p>
<p class="text-justify">Justified text</p>
```

---

### 2️⃣ **Responsive alignment**

You can make alignment change based on screen size:

```html
<p class="text-left md:text-center lg:text-right">
  Responsive alignment
</p>
```

* `text-left` on small screens
* `text-center` on medium (`md:`)
* `text-right` on large (`lg:`)

---

### 3️⃣ **Direction-based alignment**

If you’re working with RTL (right-to-left) languages, Tailwind also has:

```html
<p class="text-start">Aligns based on writing mode (start)</p>
<p class="text-end">Aligns based on writing mode (end)</p>
```


✅ **Summary**

* `text-left` → Left
* `text-center` → Center
* `text-right` → Right
* `text-justify` → Justify
* `text-start` / `text-end` → Respect text direction (LTR/RTL)
* Add **breakpoints** (`sm:`, `md:`, `lg:`, `xl:`) for responsive control.
