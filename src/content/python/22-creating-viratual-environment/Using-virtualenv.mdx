### Install the `virtualenv` package

```bash
pip install virtualenv
```

### Creating a new project

You can create a new Python project using the virtualenv command:

```bash
virtualenv "Env Name"
virtualenv venv
# or
python -m virtualenv "Env Name" 
python -m virtualenv venv 
```

### Activate the virtual environment

You can activate the virtual environment using the activate command:

```bash
# Windows
"Env Name"\Scripts\activate
```

```bash
# Linux and macOS
source "Env Name"/bin/activate
```

### Deativate the virtual environment

```bash
deactivate
```

### Install libraries

You can install libraries using the pip command:

```bash
pip install [package-name]
```
