A **dictionary** in Python is a built-in data structure that stores **key–value pairs**.  
Each **key** must be unique and immutable (such as strings, numbers, or tuples), while **values** can be of any data type—even lists or other dictionaries.

Dictionaries are ideal when you want to store data in a labeled or descriptive way.

```python
person = {
    "name": "John",
    "age": 30,
    "city": "New York"
}
````

Here:

* `"name"`, `"age"`, and `"city"` are **keys**.
* `"John"`, `30`, and `"New York"` are their corresponding **values**.

You can access values using their keys:

```python
print(person["name"])  # John
print(person["age"])   # 30
```

---

## Common Dictionary Operations

### Adding a New Key–Value Pair

```python
person["email"] = "john@example.com"
```

### Updating an Existing Value

```python
person["age"] = 31
```

### Removing a Key–Value Pair

```python
del person["city"]
```

## Dictionary Methods

Dictionaries come with several helpful methods. Below are the most commonly used ones, explained in a simple and friendly way.


### 1. `get(key)` — Safe Value Access

Instead of raising an error if a key doesn't exist, `get()` returns `None` (or a default value if provided).

```python
person = {"name": "John", "age": 30}

print(person.get("name"))   # John
print(person.get("state"))  # None
print(person["state"])      # Raises KeyError
```

This makes `get()` extremely useful when you are unsure if a key exists.

---

### 2. `pop(key)` — Remove and Return a Value

Removes the item with the given key and returns its value.

```python
person = {"name": "John", "age": 30, "city": "New York"}

age = person.pop("age")
print(age)     # 30
print(person)  # {'name': 'John', 'city': 'New York'}
```

---

### 3. `popitem()` — Remove the Last Inserted Item

Removes and returns the last added key–value pair as a tuple.

```python
person = {"name": "John", "age": 30, "city": "New York"}

last_item = person.popitem()
print(last_item)  # ('city', 'New York')
print(person)     # {'name': 'John', 'age': 30}
```

Useful when treating the dictionary like a stack (LIFO).

---

### 4. `copy()` — Create a Shallow Copy

Creates a new dictionary with the same key–value pairs.

```python
person = {"name": "John", "age": 30}

person_copy = person.copy()
print(person_copy)  # {'name': 'John', 'age': 30}
```

Note: It is a **shallow copy**, meaning nested objects are not duplicated.

---

### 5. `update()` — Merge or Overwrite Values

Adds items from another dictionary or iterable.
If the key exists, its value is replaced.

```python
person = {"name": "John", "age": 30}

person.update({"age": 31, "city": "New York"})
print(person)
# {'name': 'John', 'age': 31, 'city': 'New York'}
```

---

### 6. `keys()` — Get All Keys

Returns a dynamic view of all dictionary keys.

```python
person = {"name": "John", "age": 30}

print(person.keys())  # dict_keys(['name', 'age'])
```

---

### 7. `values()` — Get All Values

Returns a dynamic view containing all values.

```python
print(person.values())  # dict_values(['John', 30])
```

---

### 8. `items()` — Get All Key–Value Pairs

Returns pairs as tuples inside a dynamic view object.

```python
print(person.items())
# dict_items([('name', 'John'), ('age', 30)])
```

---

### 9. `clear()` — Remove Everything

Deletes all items from the dictionary:

```python
person.clear()
print(person)  # {}
```
