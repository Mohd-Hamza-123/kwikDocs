A string is just a collection of characters ‚Äî letters, numbers, symbols ‚Äî inside quotes.

```py
print("Hello World")
print('Hello World')
print("""This is 
multiline 
     string
""")
```

### Index ‚Äî What is it?

Imagine your string is a row of boxes,
and **each box holds one character**.

Example:

```
word = "Python"
```

Now think like this üëá

| Character | P | y | t | h | o | n |
| --------- | - | - | - | - | - | - |
| Index     | 0 | 1 | 2 | 3 | 4 | 5 |

So:

* `"P"` is at position **0**
* `"y"` is at position **1**
* `"n"` is at position **5**

This **position number** is called an **index**.

---

### üéØ Accessing Characters (Indexing)

You can pick any character from a string using **square brackets** `[]` with its index number.

```python
word = "Python"

print(word[0])  # Output: P
print(word[1])  # Output: y
print(word[5])  # Output: n
```

üí° In programming, counting starts from **0**, not 1.

---

### Negative Indexing (From the End)

You can also count from the end using **negative numbers**:

| Character | P  | y  | t  | h  | o  | n  |
| --------- | -- | -- | -- | -- | -- | -- |
| Index     | -6 | -5 | -4 | -3 | -2 | -1 |

So:

```python
word = "Python"

print(word[-1])  # 'n' (last letter)
print(word[-2])  # 'o'
print(word[-3])  # 'h'
```

### **String Methods :**

Here are some essential string methods in Python with examples:

### 1\. **`strip()`**

Removes leading and trailing whitespace (or specified characters).
```py
s = "  Hello World  "
b = s.strip()
print(b)  # Output: "Hello World"
```  

### 2\. **`lower()`**

Converts all characters in the string to lowercase.
```py
s = "HELLO"
b = s.lower()
print(b)  # Output: "hello"
```

### 3\. **`upper()`**

Converts all characters in the string to uppercase.
```py
s = "hello"
b = s.upper()
print(b)  # Output: "HELLO"
``` 

### 4\. **`replace(old, new)`**

Replaces occurrences of a substring with another substring.
```py
s = "Hello, World!"
b = s.replace("World", "Python")
print(b)  # Output: "Hello, Python!"
```

### 5\. **`startswith(prefix)`**

Checks if the string starts with the specified prefix.
```py
s = "Python Programming"
a = s.startswith("Python")
print(a)  # Output: True
``` 

### 6\. **`endswith(suffix)`**

Checks if the string ends with the specified suffix.
```py
s = "example.txt"
a = s.endswith(".txt")
print(a)  # Output: True
```

### 7\. **`count(substring)`**

Counts the occurrences of a substring in the string.
```py
s = "banana"
c = s.count("a")
print(c)  # Output: 3
``` 

### 8\. **`capitalize()`**

Capitalizes the first character of the string.

```py
s = "python"
c = s.capitalize()
print(c)  # Output: "Python"
```

### 9\. **`title()`**

Capitalizes the first character of every word in the string.
```py
s = "hello world"
b = s.title()
print(b)  # Output: "Hello World"
``` 

### 10\. **`isalpha()`**

Checks if all characters in the string are alphabetic(A‚ÄìZ or a‚Äìz)
```py
s = "Python"
a = s.isalpha()
print(a)  # Output: True
``` 

### 11\. **`islower()`**

Checks if all characters in the string are lowercase.
```py
s = "hello"
b = s.islower()
print(b)  # Output: True
```

### 12\. **`isupper()`**

Checks if all characters in the string are uppercase.
```py
s = "HELLO"
b = s.isupper()
print(b)  # Output: True
```

### 13\. **`isdigit()`**

It checks whether every single character in the string is a digit (0‚Äì9)

```py
s = "12345"
b = s.isdigit()
print(s)  # Output: True
```

### 14\. **`find(substring)`**

Returns the index of the first occurrence of the substring. Returns `-1` if not found.
```py
s = "Hello, World!"
b = s.find("World")
print(b)  # Output: 7
``` 

### 15\. **`split(delimiter)`**

Splits the string into a list of substrings based on a delimiter.
```py
s = "apple,banana,cherry"
print(s.split(","))  # Output: ['apple', 'banana', 'cherry']
```

### 16\. **`join(iterable)`**

Joins elements of an iterable into a single string, separated by the string used for `join`.
```py
words = ["Hello", "World"]
print(" ".join(words))  # Output: "Hello World"
```




### Escape Character ( \\ )

It is used to include special characters in a string that would otherwise be difficult or impossible to insert.

- `Escaping Quotes`: 

```python
print("He said, \"I am a python Programmer\"")
# Output: He said, "I am a python Programmer"
```

- `Including Special Characters` : Some characters have special meanings in Python strings (e.g., newline, tab). The backslash allows you to insert these characters.

```python
print("Line 1\nLine 2")  # Inserts a newline
# Output:
# Line 1
# Line 2

print("Tab\tSpace")  # Inserts a tab
# Output: Tab    Space
```

- `Using Unicode Characters:` 

```python
print("\u2764")  # Unicode for a heart symbol
# Output: ‚ù§
```

- `Escaping the Backslash Itself` 

```python
print("This is a backslash: \\")
# Output: This is a backslash: \
```

### **Slicing Strings :**

You can return a range of characters by using the slice syntax.

Specify the start index and the end index, separated by a colon, to return a part of the string.

```py
str = "0123456789"

s1 = str[:]
print(s1)    # output : 0123456789

s2 = str[1:5]
print(s2)    # output : 1234

s3 = str[4:]
print(s3)    # output : 456789

s4 = str[:9]
print(s4)    # output : 012345678 

s5 = str[0:len(str):2]
print(s5)   # output : 02468
```
