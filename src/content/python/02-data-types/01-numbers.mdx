There are three numeric types in Python:

*   *`int`*
*   *`float`*
*   *`complex`*
    
```python
x = 1        # int
y = 2.8      # float
z = 2 + 1j   # complex
```

To verify the type of any object in Python, use the `type()` function:

```python
print(type(x))
print(type(y))
print(type(z))

# output : 
# <class 'int'>
# <class 'float'>  
# <class 'complex'>
```

### **Number Functions :**

Python provides a set of built-in functions for performing numeric operations.

**1) round(x,n) Function**

round(x, n): The built-in round function rounds a number x to n decimal places. If n is not provided, it rounds to the nearest integer. 

* round() function uses Banker's Rounding (also called "round half to even").

```python
a = round(2.2)
b = round(2.8)
c = round(2.5)
f = round(3.5)
d = round(2.84919,3)
print(a) # Output: 2
print(b) # Output: 3
print(c) # Output: 2
print(f) # Output: 4
print(d) # Output: 2.849
```

**2) abs(x) Function**

abs(x): The built-in abs function returns the absolute value of a number.

```python
a = abs(-1)
b = abs(-1.1)
c = abs(5)
print(a)   # Output: 1
print(b)   # Output: 1.1
print(c)   # Output: 5
```

**3) pow(x,y) Function**

pow(x,y): The built-in pow function returns x to the power of y.

```python
a  = pow(2,3)
print(a) # Output: 8
```

**4) max(x,y) Function**

max(x,y): The built-in max function returns the largest of two or more numbers.

```python
a = max(1,2,3)
b = max(1,2,3,4)
print(a) # Output: 3
print(b) # Output: 4
```

**5) min(x,y) Function**

min(x,y): The built-in min function returns the smallest of two or more numbers.

```python    
a = min(1,2,3)
b = min(1,2,3,4)
print(a) # Output: 1
print(b) # Output: 1
```

**6) int(x) Function**

int(x): The built-in int function returns the integer part of a number.

```python
a = int(5.8)
b = int(2.5)
c = int(2.84919)
print(a) # Output: 5
print(b) # Output: 2    
print(c) # Output: 2
```

### **Python Math Module :**

Python has a built-in module that you can use for mathematical tasks. The `math` module has a set of methods and constants.

**1) math.pi**

```python
import math
print(math.pi)
```

**2) math.floor() Method**  

The `math.floor()` method returns the integer less than or equal to a number.

```python
import math
a = math.floor(0.6)   
b = math.floor(1.4)   
c = math.floor(5.3)   
d = math.floor(-5.3)  
e = math.floor(22.6)  
f = math.floor(10.0)  

print(a) # Output: 0
print(b) # Output: 1
print(c) # Output: 5
print(d) # Output: -6
print(e) # Output: 22
print(f) # Output: 10
```

**3) math.trunc() Method**

The `math.trunc()` method returns the integer part of a number.

```python
import math

a = math.trunc(5.9)   
b = math.trunc(-5.9)  
c = math.trunc(10)

print(a) # Output: 5
print(b) # Output: -5
print(c) # Output: 10
```

**4) math.ceil() Method**

The `math.ceil()` method returns the integer greater than or equal to a number.

```python
import math

a = math.ceil(0.6)
b = math.ceil(1.4)
c = math.ceil(-5.3)

print(a) # Output: 1
print(b) # Output: 2
print(c) # Output: -5
```

**5) math.sqrt() Method**

The `math.sqrt()` method returns the square root of a number.

```python
import math

a = math.sqrt(4)
b = math.sqrt(9)

print(a) # Output: 2.0
print(b) # Output: 3.0

```

**6) math.pow() Method**

The `math.pow()` method returns the value of a number raised to the power of another number.

```python
import math

a = math.pow(2, 3)

print(a) # Output: 8
```

**7) math.factorial() Method**

The `math.factorial()` method returns the factorial of a number.

```python
import math

a = math.factorial(5)
print(a) # Output: 120
```

### **Python Random Module :**

** 1) random.randint( ) Method**

The `random.randint()` method returns a random integer between the specified range.

```python
import random

a = random.randint(3, 9)
print(a) # returns a random number between 3 and 9 (both included)
```

**2) random.choice( ) Method**

The `random.choice()` method returns a random element from a list.

```python
import random
mylist = ["apple", "banana", "cherry"]
print(random.choice(mylist))
```

### **Python Fractions Module :**

This module provides support for rational number arithmetic.

**1) Fraction( ) Method :**

```python
from fractions import Fraction

a = Fraction('0.5')
c = Fraction('0.333')

print(a) # Output: 1/2
print(c) # Output: 333/1000
```

```python
from fractions import Fraction

a = Fraction(1, 2)
b = Fraction(1, 3)

print(a)  # Output: 1/2
print(b)  # Output: 1/3
```

### **Boolean as 0 , 1 :**

In Python, Boolean values (True and False) are treated as integers:

- True behaves like 1
- False behaves like 0

```python
print(0 == False) # True
print(1 == True) # True

print(0 is False) # False
print(1 is True)  # False
```
