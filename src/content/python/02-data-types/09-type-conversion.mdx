Type conversion in Python refers to the process of converting one data type into another. Python supports both **implicit** (automatic) and **explicit** (manual) type conversion. Let's explore how this works! ğŸš€

## Implicit Type Conversion (Automatic) ğŸ¤–

Python automatically converts one data type to another without any user involvement. This usually happens when you perform operations between different data types.

```python
# Example of implicit type conversion
num_int = 10          # Integer
num_float = 5.5       # Float

result = num_int + num_float  # Python automatically converts num_int to float
print(result)  # Output: 15.5
print(type(result))  # Output: <class 'float'>
```

## Explicit Type Conversion (Manual) ğŸ› ï¸

Explicit type conversion, also known as **type casting**, requires the user to manually convert one data type to another using built-in functions like `int()`, `float()`, `str()`, etc.

### 1) Converting to Integer (`int()`) ğŸ”¢

```python
num_str = "10"
num_int = int(num_str)  # Convert string to integer
print(num_int)  # Output: 10
print(type(num_int))  # Output: <class 'int'>
```

### 2) Converting to Float (`float()`) ğŸ¯

```python
num_str = "10.5"
num_float = float(num_str)  # Convert string to float
print(num_float)  # Output: 10.5
print(type(num_float))  # Output: <class 'float'>
```

### 3) Converting to String (`str()`) ğŸ”¤

```python
num_int = 10
num_str = str(num_int)  # Convert integer to string
print(num_str)  # Output: "10"
print(type(num_str))  # Output: <class 'str'>
```

### 4) Converting to List (`list()`) ğŸ“œ

```python
my_tuple = (1, 2, 3)
my_list = list(my_tuple)  # Convert tuple to list
print(my_list)  # Output: [1, 2, 3]
print(type(my_list))  # Output: <class 'list'>
```

### 5) Converting to Tuple (`tuple()`) ğŸ“¦

```python
my_list = [1, 2, 3]
my_tuple = tuple(my_list)  # Convert list to tuple
print(my_tuple)  # Output: (1, 2, 3)
print(type(my_tuple))  # Output: <class 'tuple'>
```

### 6) Converting to Set (`set()`) ğŸ²

```python

my_set = set(my_list)  # Convert list to set (removes duplicates)
print(my_set)  # Output: {1, 2, 3}
print(type(my_set))  # Output: <class 'set'>
```

### 7) Converting to Dictionary (`dict()`) ğŸ“š

```python

my_dict = dict(my_list)  # Convert list of tuples to dictionary
print(my_dict)  # Output: {'a': 1, 'b': 2, 'c': 3}
print(type(my_dict))  # Output: <class 'dict'>
```

### 8) Converting to Boolean (`bool()`) âœ”ï¸

```python

```

### 9) Converting to Bytes (`bytes()`) ğŸ”¢

```python

```

### 10) Converting to Complex (`complex()`) ğŸŒ€

```python

```

---

## Key Points to Remember ğŸ§ 

- Implicit type conversion is performed automatically by Python.
- Explicit type conversion requires the use of built-in functions like `int()`, `float()`, `str()`, etc.
- Not all types can be converted to each other. For example, converting a string like `"hello"` to an integer will raise a `ValueError`.
- Be cautious when converting between types, as data loss or errors may occur (e.g., converting a float to an integer truncates the decimal part).
