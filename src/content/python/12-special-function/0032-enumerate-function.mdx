Enumerate function is used to iterate over a sequence and return the index and the value of the sequence.

*Example 1* :
```python
>>> for index, value in enumerate(['a', 'b', 'c']):
...     print(index, value)
# output :
# 0 a
# 1 b
# 2 c
```

*Example 2* :
```python
for index,num in enumerate(range(5,10)):
    print(f"index = {index}, num = {num}")
# output :
# index = 0, num = 5
# index = 1, num = 6
# index = 2, num = 7
# index = 3, num = 8
# index = 4, num = 9
```
