Inheritance is a mechanism where a new class inherits properties and methods from an existing class. This promotes code reuse.

The basic syntax for inheritance in Python is as follows:

```python
class ParentClass:
    # Parent class code
    pass

class ChildClass(ParentClass):
    # Child class code
    pass
```

#### Example 1 : 

```python 
class Car:
    def __init__(self,brand,model):
        self.brand = brand
        self.model = model 

    def details(self):
        return f"brand = {self.brand}, model = {self.model}"
    
class ElectricCar(Car):
    def __init__(self,batterySize):
        self.batterySize = batterySize
tesla = ElectricCar('200kwh')
print(tesla.__dict__) # {'batterySize': '200kwh'}
```

In above example ElectricCar class should inherit parent attributes (brand , model ). To get access we use `super` method in Child class. 

#### Example 2 :

```python showLineNumbers {11}
class Car:
    def __init__(self,brand,model):
        self.brand = brand
        self.model = model 

    def details(self):
        return f"brand = {self.brand}, model = {self.model}"

class ElectricCar(Car):
    def __init__(self,brand,model,batterySize):
        super().__init__(brand,model)
        self.batterySize = batterySize

tesla = ElectricCar("Tesla","Model-RV","200kwh")
print(tesla.__dict__) # {'brand': 'Tesla', 'model': 'Model-RV', 'batterySize': '200kwh'}
```
