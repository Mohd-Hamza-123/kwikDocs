```html
    <!DOCTYPE html> 
    <!-- This tells the browser which version of HTML is being used. -->
    <html lang="en">
    <!--  It says , language of the content as English. -->
    <head>
        <!-- The "head" section contains meta-information 
    about the document, such as title, styles, and scripts. -->
        <meta charset="UTF-8">
        <!-- This tells the browser how to read and display text,
     so all letters, symbols, and characters show up correctly. -->
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <!-- Makes the website responsive. -->
        <title>title</title>
        <!-- Sets the title of the web page, which appears in the browser tab. -->
    </head>
    <body>
        <!-- The "body" section contains the main content of the 
    web page that will be displayed to users. -->
    </body>
    </html>
    <!-- Closes the HTML document. -->
```

- `<!DOCTYPE html>` : Declares the HTML version.

- `<html lang="en">` : Root tag, sets the language to English.

- `<head>` : Holds page metadata like title and styles.

- `<meta charset="UTF-8">` : Ensures proper display of text and symbols.

- `<meta name="viewport" content="width=device-width, initial-scale=1.0">` : Makes the page responsive.

- `<title>` : Sets the page title in the browser tab.

- `<body>` : Contains the visible content of the page.
