The `<a>` tag defines a hyperlink, which is used to link from one page to another.
```html
    <!-- Syntax -->
    <a href="link" >click here</a>
```
**Absolute Link :**
```html
<a href="https://thoughtify.vercel.app" target="_self" >Visit<u>Thoughtify</u></a>
<a href="https://www.w3schools.com" target="_blank">Visit Web site</a>
```
**Relative Link :**
```html
<a href="./page1.html">Visit page</a>
<a href="../page2.html">Visit page</a>  
<a href="../../page3.html">Visit page</a>
```
The `target` attribute specifies where to open the linked document.

*   `_self` -   Default. Opens the document in the same window/tab as it was clicked
*   `_blank` -   Opens the document in a new window or tab
