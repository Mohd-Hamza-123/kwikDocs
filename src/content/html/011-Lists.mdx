In HTML, lists are used to organize and display items in a structured format.

Ordered lists are used when the order of items matters. The `<ol>` tag creates a numbered list, and the `type` attribute specifies the type of numbering.

<HtmlEditor id={1} defaultCode={`<ol type="1">
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
  <li>Fourth item</li>
  <li>Fifth item</li>
</ol>`}/>

#### `type` Attribute
The `type` attribute specifies the kind of marker to use in the list:

- `1`: Numbers (default)
- `A`: Uppercase letters
- `a`: Lowercase letters
- `I`: Uppercase Roman numerals
- `i`: Lowercase Roman numerals
