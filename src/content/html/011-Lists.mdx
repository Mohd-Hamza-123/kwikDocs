In HTML, lists are used to organize and display items in a structured format.

Ordered lists are used when the order of items matters. The `<ol>` tag creates a numbered list, and the `type` attribute specifies the type of numbering.

#### Syntax
```html
<ol type="1">
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>
```

#### `type` Attribute
The `type` attribute specifies the kind of marker to use in the list:

- `1`: Numbers (default)
- `A`: Uppercase letters
- `a`: Lowercase letters
- `I`: Uppercase Roman numerals
- `i`: Lowercase Roman numerals

For more details, refer to the [type attribute documentation](https://www.w3schools.com/tags/att_ol_type.asp).

Unordered lists are used when the order of items does not matter. The `<ul>` tag creates a bulleted list, and the `type` attribute specifies the style of the bullets.

#### Syntax
```html
<ul type="disc">
  <li>Apples</li>
  <li>Bananas</li>
  <li>Oranges</li>
</ul>
```
