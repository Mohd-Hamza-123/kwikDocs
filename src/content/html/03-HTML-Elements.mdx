HTML elements are the building blocks of an HTML document. They define the structure and content of a web page.

An element typically consists of:

* **Start tag**: Indicates the beginning of the element.
* **Content**: The content to be displayed or processed.
* **End tag**: Closes the element (not required for self-closing elements).


#### **Syntax of HTML Elements:**

```html
<tagname>Content</tagname>
```

**Example :**

```html
<p>This is a paragraph.</p>
<h1>This is a heading.</h1>
```

---

```html
<tagname attribute="value">Content</tagname>
```

**Example :**

```html
<p class="top">This is a paragraph with a class attribute.</p>
<h3 id="some-id">This is a heading with an id attribute.</h3>
```

---

```html
<tagname attribute1="value1" attribute2="value2">Content</tagname>
```

**Example :**

```html
<p style="color: red;" class="center">This is a paragraph with a style attribute.</p>
<h1 id="primary" class="box">This is a heading with a data attribute.</h1>
```

---

```html
<tagname attribute="value">
```

**Example :**

```html
<input type="text">
<input type="email" placeholder="Enter your email">
```

### **Types of HTML Elements**

### **1. Non-Self-Closing (Container) Tags**

These elements must have an opening and a closing tag.

* **Paragraph (`<p>`)**: Defines a paragraph.

<HtmlEditor id={1} defaultCode={`<p>This is a paragraph.</p>`}/>

* **Heading (`<h1>` to `<h6>`)**: Defines headings.

<HtmlEditor id={2} defaultCode={`
<html>
<h1>Main Heading</h1>
<h2>Subheading 2</h2>
<h3>Subheading 3</h3>
<h4>Subheading 4</h4>
<h5>Subheading 5</h5>
<h6>Subheading 6</h6>
</html>
`}/>

### **2. Self-Closing (Void) Tags**

Elements that do not require a closing tag.

* **Image (`<img>`)**:

<HtmlEditor id={3} defaultCode={`<html lang="en">
<body>
  <img src="https://images.pexels.com/photos/29914981/pexels-photo-29914981.jpeg" alt="cameraman" width="200" height="200"/>
</body>
</html>`}/>

* **Line Break (`<br>`)**:

<HtmlEditor id={4} defaultCode={`<p>First line<br>Second line</p>`}/>

* **Meta (`<meta>`)**:

```html
<meta charset="UTF-8">
```
