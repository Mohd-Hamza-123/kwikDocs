### **SuperScript**

The `<sup>` tag defines superscript text.

<HtmlEditor id={1} defaultCode={`<p>This text contains <sup>superscript</sup> text</p>
<p>(a + b)<sup>2</sup> = a<sup>2</sup> + b<sup>2</sup> + 2ab</p>
`}/>


### **SubScript**

The `<sub>` tag defines subscript text.

<HtmlEditor id={2} defaultCode={`<p>This text contains <sub>subscript</sub> text</p>
<p>H<sub>2</sub>O + O = H<sub>2</sub>O</p>
`}/>
