---
title: Creating and Deleting Collection
published: true
---


### 1. See collections

```js
show collections
```

---

### 2. Create a collection

```js
db.createCollection("users")
```

(or auto-created on first insert)

---

### 3. Drop collection

```js
db.<collection_name>.drop()
db.users.drop()
```


Clear screen:

```js
cls    // Windows
clear  // Linux/macOS
```

Exit shell:

```js
exit
```
