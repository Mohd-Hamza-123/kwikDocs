## üìÑ Document (smallest unit)

A **document** is:

* A single data record
* Stored in **BSON** (binary JSON)
* Flexible (no fixed schema)

Example document:

```json
{
  "_id": "u123",
  "name": "Hamza",
  "age": 24,
  "skills": ["Node.js", "MongoDB"],
  "isActive": true
}
```

Think of a document as:

* One **row** in SQL
* One **object** in JavaScript
* One **JSON file**
