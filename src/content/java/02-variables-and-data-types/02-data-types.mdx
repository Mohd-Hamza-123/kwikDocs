### ðŸ”µ Non-Primitive Data Types

Non-primitive types are **reference types**. They refer to objects and have more complex structures.

| Type      | Description                                  | Default Value |
|-----------|----------------------------------------------|----------------|
| `String`  | Sequence of characters                       | `null`         |
| `Array`   | Collection of similar data types             | `null`         |
| `Class`   | Blueprint for creating objects               | `null`         |
| `Interface`| Reference to interface methods              | `null`         |
| `Object`  | Root class of all Java classes               | `null`         |

> ðŸ”¸ All non-primitive types are stored by **reference** (not by value) and can be assigned `null`.

### âœ… Summary

- **Primitive types**: Simple, fixed-size data like `int`, `char`, `boolean`
- **Non-Primitive types**: Complex structures like `String`, `Array`, and user-defined types
- Primitive types have **default values** and fixed sizes; non-primitives are **reference types** and default to `null`
