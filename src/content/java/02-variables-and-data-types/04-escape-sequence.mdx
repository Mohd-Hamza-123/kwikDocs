In Java, escape sequences are used to represent special characters in strings.

```java
public class EscapeExample {
    public static void main(String[] args) {
        System.out.println("Line1\nLine2");           // New line
        System.out.println("Hello\tWorld");           // Tab
        System.out.println("He said, \"Java is fun\""); // Quotes inside string
        System.out.println("Backslash: \\");          // Backslash
        System.out.println("Unicode A: \u0041");      // Unicode for 'A'
    }
}

```
