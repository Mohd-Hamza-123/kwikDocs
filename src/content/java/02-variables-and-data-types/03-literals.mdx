In Java, **literals** are the fixed values assigned directly to variables or used in expressions.

### ðŸ”¹ Types of Literals in Java

| Type                       | Example                        | Description                                                                     |
| -------------------------- | ------------------------------ | ------------------------------------------------------------------------------- |
| **Integer literal**        | `10`, `-50`, `0b1010`, `0x1A`  | Whole numbers (decimal, binary `0b`, octal `0`, or hexadecimal `0x`)            |
| **Floating-point literal** | `3.14`, `-0.01`, `1e3`, `2.5f` | Decimal numbers. Add `f` or `F` for `float`, `d` or `D` (optional) for `double` |
| **Character literal**      | `'A'`, `'9'`, `'@'`, `'\n'`    | Single characters enclosed in single quotes                                     |
| **String literal**         | `"Hello"`, `"123"`, `""`       | Sequence of characters enclosed in double quotes                                |
| **Boolean literal**        | `true`, `false`                | Only two possible values                                                        |
| **Null literal**           | `null`                         | Represents the null reference (used for objects)                                |
