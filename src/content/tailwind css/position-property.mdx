### ðŸŽ¯ **2. Offset utilities (top, right, bottom, left)**

Use `inset-*` or `top-*`/`right-*`/`bottom-*`/`left-*`.

```html
<div class="absolute top-0 left-0">Top left</div>
<div class="absolute bottom-0 right-0">Bottom right</div>
```

* `top-0`, `right-0`, `bottom-0`, `left-0` â†’ stick to edges
* `top-4`, `left-10` â†’ offsets using Tailwind spacing scale
* `inset-0` â†’ set `top: 0; right: 0; bottom: 0; left: 0;`
* `inset-x-0` â†’ left & right = 0
* `inset-y-0` â†’ top & bottom = 0
