### ðŸŽ¨ **2. Axis-specific overflow**

You can control **X (horizontal)** and **Y (vertical)** separately:

```html

<div class="overflow-y-scroll">Always vertical scroll</div>
<div class="overflow-x-hidden">Hide horizontal overflow</div>
<div class="overflow-y-visible">Show vertical overflow</div>
```

---

**3. Example**

```html
<div class="w-40 h-20 border overflow-auto">
  <p>This is some long text that will scroll inside the box...</p>
</div>
```
