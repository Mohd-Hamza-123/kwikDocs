HTTP methods are the standardized commands used to interact with web servers. They include:

### GET Method

- **GET**: Retrieves data from a server.
- **Use Case**: Fetching a web page or an API resource.
Eg : When you type a URL in the browser's address bar and press Enter, the browser issues a GET request.
