Dockerizing a Node.js app helps you package everything (code, dependencies, runtime) into a single container, making it easy to run anywhere.


### üß± Step 1: Create a `Dockerfile`

Place this `Dockerfile` in the **root folder** of your Node.js project:

```Dockerfile
# Use the official Node.js 22 slim image as base (lightweight Linux + Node)
FROM node:22-slim

# Set the working directory inside the container to /app
WORKDIR /app

# Copy package.json and package-lock.json to install dependencies
COPY package.json package.json
COPY package-lock.json package-lock.json

# Install dependencies
RUN npm install

# Copy the rest of the application code to /app
COPY . .

# Expose port 3000 (the port your app listens on)
EXPOSE 3000

# Define the command to run the app
CMD ["npm", "start"]
```

---

### üõ†Ô∏è Step 2: Build the Docker Image

Open your terminal in the project directory (where the `Dockerfile` is located) and run:

```bash
docker build -t my-node-app .
```

### üí° Explanation:

* `docker build`: Command to build a Docker image.
* `-t my-node-app`: Tags the image with a name (`my-node-app`) for easy reference.
* `.`: Tells Docker to use the **current directory** (where your `Dockerfile` and app files are located).

---

### ‚ñ∂Ô∏è Step 3: Run the Docker Container

Once the image is built, run the container:

```bash
docker run -p 3000:3000 my-node-app
```

### üí° Explanation:

* `docker run`: Runs a container from an image.
* `-p 3000:3000`: Maps **port 3000 of the container** to **port 3000 on your host machine**.
* `my-node-app`: The name of the image you created earlier.

### üîç Access Your App:

Open your browser and visit:
[http://localhost:3000](http://localhost:3000)

### Hot Reloading

For Hot Reloading, you can use `nodemon` in your `package.json`:

```json
"scripts": {
  "start": "nodemon index.js" // in Linux 
}
```

```json
"scripts": {
      "start": "nodemon --legacy-watch index.js" // in windows
}
```

Then run:

```bash
docker run -p 3000:3000 -v ${PWD}:/app node:22-slim sh -c "cd /app && npm install && npm start"
```

### üß∞ Helpful Docker Commands

| Purpose                  | Command                      |
| ------------------------ | ---------------------------- |
| List Docker images       | `docker images`              |
| List running containers  | `docker ps`                  |
| List all containers      | `docker ps -a`               |
| Stop a container         | `docker stop <container_id>` |
| Remove a container       | `docker rm <container_id>`   |
| Remove an image          | `docker rmi <image_name>`    |
| View logs of a container | `docker logs <container_id>` |
