### **Syntax for Inserting Data**

```sql
INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...);
```

---

### **Inserting Data in MySQL and PostgreSQL**

```sql
INSERT INTO employees (name, position, department, hire_date, salary)
VALUES ('Mohd Hamza', 'Software Engineer', 'Senior', '2025-05-22', '120000.00'),
       ('Mohd Tariq', 'Software Tester', 'Senior', '2025-05-22', '12000.00'),
       ('Mohd Quds', 'Data Analyst', 'Senior', '2025-05-22', '100000.00'),
       ('Mohd Hinata', 'Accountant', 'Junior', '2025-05-22', '110000.00');
```
