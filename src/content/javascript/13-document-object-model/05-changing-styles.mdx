You can dynamically change the style of HTML elements using JavaScript in a few different ways.

---

### ✅ 1. **Using `style` Property**

Directly access and set inline CSS styles using the `.style` property.

**Example:**
```js
let box = document.getElementById("myBox");
box.style.backgroundColor = "lightblue";
box.style.fontSize = "20px";
```

> Note: Property names use camelCase (e.g. `backgroundColor`, `fontSize`) instead of CSS syntax (`background-color`, `font-size`).

---

### ✅ 2. **Modifying `className`**

You can change all classes at once using `className`.

**Example:**

```js
let box = document.getElementById("myBox");
box.className = "highlighted";
```

---

### ✅ 3. **Using `classList` Methods**

For more control over CSS classes, use `classList`:

#### ➤ `add()` – adds a class

```js
element.classList.add("active");
```

#### ➤ `remove()` – removes a class

```js
element.classList.remove("hidden");
```

#### ➤ `toggle()` – adds/removes class depending on whether it's present

```js
element.classList.toggle("dark-mode");
```

#### ➤ `contains()` – checks if class exists

```js
if (element.classList.contains("error")) {
  alert("There is an error.");
}
```
