To **access elements in the DOM using JavaScript**, you use methods provided by the `document` object. These let you **select HTML elements** so you can read or change them.


### ðŸ“Œ Common Ways to Access Elements

#### ðŸ”¹ 1. `getElementById()`

* Selects a single element by its `id`.

```html
<p id="greet">Hello</p>
```

```js
const para = document.getElementById("greet");
console.log(para.textContent); // "Hello"
```

---

#### ðŸ”¹ 2. `getElementsByClassName()`

* Selects **all elements** with a specific class.
* Returns an **HTMLCollection** (like an array, but not exactly).

```html
<p class="info">Para 1</p>
<p class="info">Para 2</p>
```

```js
const paras = document.getElementsByClassName("info");
console.log(paras[0].textContent); // "Para 1"
```

---

#### ðŸ”¹ 3. `getElementsByTagName()`

* Selects all elements with a given tag name.

```html
<h1>Title</h1>
<h1>Heading</h1>
```

```js
const headings = document.getElementsByTagName("h1");
console.log(headings[1].textContent); // "Heading"
```

---

#### ðŸ”¹ 4. `querySelector()`

* Returns the **first matching** element (can use CSS selectors).

```html
<p class="msg">Hi</p>
```

```js
const msg = document.querySelector(".msg");
console.log(msg.textContent); // "Hi"
```

---

#### ðŸ”¹ 5. `querySelectorAll()`

* Returns **all matching** elements (NodeList, like an array).

```html
<p class="msg">Hi</p>
<p class="msg">Bye</p>
```

```js
const allMsgs = document.querySelectorAll(".msg");
allMsgs.forEach(p => console.log(p.textContent)); 
```
