### **Function Scope**

* A variable is function scoped if it is only accessible inside the function where it is declared.
*  Only functions create function scope.


#### Ways to Construct Function Scope

```javascript
function testFunc() {
    // function scope
}
```

```javascript
const greet = function() {
  // function scope
};
```

```javascript
class Person {
  constructor(name) {
    // function scope
  }

  greet() {
    // function scope
  }
}
```

### **Block Scope**

* A variable is block scoped if it is only accessible inside the block (like `{}`) where it's declared.
* Variables declared with `let` and `const` are block scoped.

#### Ways to Construct Blocks Scope

| Construct             | Example                               | Description                                                        |
| --------------------- | ------------------------------------- | ------------------------------------------------------------------ |
| **Standalone block**  | `{ ... }`                             | Just curly braces on their own                                     |
| **`if` statement**    | `if (condition) { ... }`              | Conditional block                                                  |
| **`else` block**      | `else { ... }`                        | Paired with `if`                                                   |
| **`for` loop**        | `for (let i = 0; i < 5; i++) { ... }` | Loop block                                                         |
| **`for...of` loop**   | `for (let item of list) { ... }`      | Loop block                                                         |
| **`for...in` loop**   | `for (let key in obj) { ... }`        | Loop block                                                         |
| **`while` loop**      | `while (condition) { ... }`           | Loop block                                                         |
| **`do...while` loop** | `do { ... } while (condition)`        | Loop block                                                         |
| **`switch` block**    | `switch(x) { case 1: { ... } }`       | Each `case` block can scope variables                              |
| **`try` block**       | `try { ... }`                         | Error-handling block                                               |
| **`catch` block**     | `catch (e) { ... }`                   | Has its own block scope for `e` and inner code                     |
| **`finally` block**   | `finally { ... }`                     | Runs after try/catch                                               |
| **`class`**           | `class Person { ... }`                | Class block                                                        |

---

### **Important Points**

- Functions are function scopes, and also act as block scopes for let and const,
- A function creates `Function scope` for variables declared with var
- A function also create `Block scope` for variables declared with let and const (because the function body is a block)

```javascript
function test() {
  var x = 1;      // ✅ function scoped
  let y = 2;      // ✅ block scoped (block = function body)
  const z = 3;    // ✅ block scoped (same as above)

  console.log("Inside function:");
  console.log("x:", x); // 1
  console.log("y:", y); // 2
  console.log("z:", z); // 3
}

test();

console.log("Outside function:");
console.log("x:", typeof x); // undefined ❌ x is not defined
console.log("y:", typeof y); // undefined ❌ y is not defined
console.log("z:", typeof z); // undefined ❌ z is not defined
```

### **Difference between Function and Block Scope**

Certainly! Here's the table:

| **Feature**             | **Block Scope**                          | **Function Scope**                             |
| ----------------------- | ---------------------------------------- | ---------------------------------------------- |
| **Created by**          | Any `{}` block (like `if`, `for`, etc.)  | Only function bodies (`function`, `=>`, etc.)  |
| **Applies to**          | `let`, `const`                           | `var` (only!)                                  |
| **Used in**             | `{}`, `if`, `for`, `while`, `try`, etc.  | `function`, arrow functions, class methods     |
| **Variable visibility** | Limited to the block `{}` where declared | Limited to the entire function body            |
| **Ignored by**          | `var` (it leaks out)                     | `let`, `const` (they still follow block rules) |
