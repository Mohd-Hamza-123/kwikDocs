In JavaScript, **truthy** and **falsy** values refer to how values are interpreted in a boolean context.

A **falsy** value is one that evaluates to `false` when converted to a boolean.

*   `false`
*   `0`, `-0` and `0n`
*   `""`, `''` (empty strings)
*   `null`, `undefined` and `NaN`
*   `document.all`

    if (!false) console.log("Falsy: false");
    if (!0) console.log("Falsy: 0");
    if (!"") console.log("Falsy: empty string");
    if (!null) console.log("Falsy: null");
    if (!undefined) console.log("Falsy: undefined");
    if (!NaN) console.log("Falsy: NaN");
    

A **truthy** value is anything that is **not falsy**. Common truthy values include:

*   Non-zero numbers (e.g., `1`, `-1`, `3.14`).
*   Non-empty strings (e.g., `"hello"`, `"false"`).
*   Objects (e.g., `{}`, `[]`).
*   `true`

    if (true) console.log("Truthy: true");
    if (1) console.log("Truthy: 1");
    if ("hello") console.log("Truthy: non-empty string");
    if ({}) console.log("Truthy: empty object");
    if ([]) console.log("Truthy: empty array");
