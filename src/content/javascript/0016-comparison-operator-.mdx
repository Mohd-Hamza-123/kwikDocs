In JavaScript, comparison operators are used to compare two values and return a boolean (`true` or `false`) based on the comparison. Here are the main comparison operators:

**1) Equality (`==`)**: Compares two values for equality, converting types if necessary (loose equality).  
Example: `5 == '5'` → `true`

    // 1. Equality (==)
    console.log(5 == '5'); // true (loose equality, type conversion happens)
    console.log(5 == 5);   // true

**2) Strict Equality (`===`)**: Compares both the value and the type for equality (no type conversion).  
Example: `5 === '5'` → `false`

    // 2. Strict Equality (===)
    console.log(5 === '5'); // false (strict equality, no type conversion)
    console.log(5 === 5);   // true

**3) Inequality (`!=`)**: Checks if two values are not equal, with type conversion.  
Example: `5 != '6'` → `true`

    // 3. Inequality (!=)
    console.log(5 != '6'); // true (type conversion happens)
    console.log(5 != '5'); // false (values are the same after conversion)

**4) Strict Inequality (`!==`)**: Checks if values or types are not equal.  
Example: `5 !== '5'` → `true`

    // 4. Strict Inequality (!==)
    console.log(5 !== '5'); // true (type and value are different)
    console.log(5 !== 5);   // false (both type and value are the same)

**5) Greater than (`>`)**: Checks if the left value is greater than the right.  
Example: `7 > 5` → `true`

    // 5. Greater Than (>)
    console.log(7 > 5); // true (7 is greater than 5)
    console.log(5 > 7); // false

**6) Less than (`<`)**: Checks if the left value is less than the right.  
Example: `3 < 5` → `true`

    
    // 6. Less Than (<)
    console.log(3 < 5); // true (3 is less than 5)
    console.log(5 < 3); // false

**7) Greater than or equal to (`>=`)**: Checks if the left value is greater than or equal to the right.  
Example: `5 >= 5` → `true`

    // 7. Greater Than or Equal To (>=)
    console.log(5 >= 5); // true (5 is equal to 5)
    console.log(7 >= 5); // true (7 is greater than 5)

**8) Less than or equal to (`<=`)**: Checks if the left value is less than or equal to the right.  
Example: `3 <= 5` → `true`

    // 8. Less Than or Equal To (<=)
    console.log(3 <= 5); // true (3 is less than 5)
    console.log(5 <= 5); // true (5 is equal to 5)
