Mathematical Operators In JS :

*   _Addition (`+`)_
*   _Subtraction (`-`)_
*   _Multiplication (`*`)_
*   _Division (`/`)_
*   _Modulus (Remainder) (`%`)_
*   _Exponentiation (`**`)_
*   _Increment (`++`)_
*   _Decrement (`--`)_

*   **Modulus (Remainder) `% :`** The modulus operator returns the remainder of the division of two numbers.

    let a = 10;
    let b = 3;
    let remainder = a % b; // Remainder when 10 is divided by 3
    console.log(remainder); // Output: 1
    

*   **Exponentiation `** :`** The exponentiation operator raises the base to the power of the exponent.

    let base = 2;
    let exponent = 3;
    let result = base ** exponent; // 2 raised to the power of 3
    console.log(result); // Output: 8
    

*   **Increment (`++`) :** The increment operator increases the value of a variable by 1.

#### **Post-Increment**

The value is returned **before** incrementing.

    let a = 5;
    console.log(a++); // Output: 5 (returns value before increment)
    console.log(a);   // Output: 6 (value after increment)
    

#### **Pre-Increment**

The value is incremented **before** being returned.

    let b = 5;
    console.log(++b); // Output: 6 (increments first, then returns)
    console.log(b);   // Output: 6

*   **Decrement (`--`)  :** The decrement operator decreases the value of a variable by 1.

#### **Post-Decrement**

The value is returned **before** decrementing.

    let x = 5;
    console.log(x--); // Output: 5 (returns value before decrement)
    console.log(x);   // Output: 4 (value after decrement)
    

#### **Pre-Decrement**

The value is decremented **before** being returned.

    let y = 5;
    console.log(--y); // Output: 4 (decrements first, then returns)
    console.log(y);   // Output: 4
    

Note : Increment and decrement operators (`++` and `--`) can only be used on variables, not on constants or literal values.

**Methods on Math Object** 

1) Math.abs(x) : Returns the absolute value of a number. 

    console.log(Math.abs(-5))    // 5
    console.log(Math.abs(-4.4))  // 4.4
    console.log(Math.abs(0))     // 0
    console.log(Math.abs(-0.1))  // 0.1
    console.log(Math.abs(3))     // 3

2) Math.round(x) : Rounds a number to the nearest integer.

    console.log(Math.round(2.2)) // 2
    console.log(Math.round(3.9)) // 4
    console.log(Math.round(5.1)) // 5
    console.log(Math.round(-1.1)) // -1
    console.log(Math.round(-1.5)) // -1

3) Math.ceil(x) : Rounds a number to the greatest integer.

    console.log(Math.ceil(4.2)) // 5
    console.log(Math.ceil(-4.2)) // -4

4) Math.floor(x) : Rounds a number to the nearest integer less than or equal to a given number.

    console.log(Math.floor(5.95));
    // Expected output: 5
    
    console.log(Math.floor(5.05));
    // Expected output: 5
    
    console.log(Math.floor(5));
    // Expected output: 5
    
    console.log(Math.floor(-5.05));
    // Expected output: -6

4) Math.sqrt(x) : Gives square root of a input number 

    console.log(Math.sqrt(2)) // 1.4142135623730951

5) Math.pow(x,y) : x to the power y 

    console.log(Math.pow(4,2)) // 16

6) Math.random() : Generates a floating-point number between **0 (inclusive)** and **1 (exclusive)**.
