Here are some useful JavaScript object methods with explanations and examples:


### **Object.assign()**

**Description**: Copies the values of all enumerable properties from one or more source objects to a target object. It returns the target object.

```js
const obj1 = { name: 'Hamza', phoneNo: 4939339 };
const obj2 = { email: 'example@gmail.com' };
const obj3 = Object.assign(obj1, obj2);

console.log(obj3);
// Output: { name: 'Hamza', phoneNo: 4939339, email: 'example@gmail.com' }
```

---

### **Object.create()**

-  It is used to create a new object with the specified prototype object.  

```js
const proto = {
  greet: function () {
    return 'Hello!';
  },
};
// Create a new object with `proto` as its prototype
const obj = Object.create(proto);
console.log(obj.greet()); // Output: Hello!
// Adding properties to the new object
obj.name = 'Hamza';
console.log(obj.name); // Output: Hamza
```

---

### **Object.keys()**

**Description**: Returns an array of a given object's own enumerable property names.

```js
const obj = { name: 'Hamza', age: 25 };

console.log(Object.keys(obj));
// Output: ['name', 'age']
```

---

### **Object.values()**

**Description**: Returns an array of a given object's own enumerable property values.

```js
const obj = { name: 'Hamza', age: 25 };

console.log(Object.values(obj));
// Output: ['Hamza', 25]
```

---

### **Object.fromEntries()**

**Description**: Converts a list of key-value pairs into an object.

```js
const entries = [['name', 'Hamza'], ['age', 25]];

const obj = Object.fromEntries(entries);

console.log(obj);
// Output: { name: 'Hamza', age: 25 }
```

---

### **Object.groupBy()** *(Experimental)*

**Description**: Groups array elements by a specified criterion. *(Only available in modern environments.)*

```js
const array = [1.2, 2.4, 3.5, 4.7];

const grouped = Object.groupBy(array, Math.floor);

console.log(grouped);
// Output: { '1': [1.2], '2': [2.4], '3': [3.5], '4': [4.7] }
```

---

### **Object.preventExtensions()**

**Description**: Prevents new properties from being added to an object.

```js
const obj = { name: 'Hamza' };
Object.preventExtensions(obj);

obj.age = 25; // This will fail in strict mode
console.log(obj);
// Output: { name: 'Hamza' }
```

---

### **Object.isExtensible()**

**Description**: Checks if an object is extensible (i.e., new properties can be added).

```js
const obj = { name: 'Hamza' };

console.log(Object.isExtensible(obj)); // Output: true

Object.preventExtensions(obj);
console.log(Object.isExtensible(obj)); // Output: false
```

---

### **Object.seal()**

**Description**: Seals an object, preventing new properties from being added and marking all existing properties as non-configurable.

```js
const obj = { name: 'Hamza' };
Object.seal(obj);

obj.age = 25; // This will fail in strict mode
delete obj.name; // This will also fail

console.log(obj);
// Output: { name: 'Hamza' }
```

---

### **Object.isSealed()**

**Description**: Checks if an object is sealed.

```js
const obj = { name: 'Hamza' };

console.log(Object.isSealed(obj)); // Output: false

Object.seal(obj);
console.log(Object.isSealed(obj)); // Output: true
```

---

### **Object.freeze()**

**Description**: Freezes an object, making it immutable. No properties can be added, removed, or modified.

```js
const obj = { name: 'Hamza' };
Object.freeze(obj);

obj.name = 'Ali'; // This will fail in strict mode

console.log(obj);
// Output: { name: 'Hamza' }
```

---

### **Object.isFrozen()**

**Description**: Checks if an object is frozen.

```js
const obj = { name: 'Hamza' };

console.log(Object.isFrozen(obj)); // Output: false

Object.freeze(obj);
console.log(Object.isFrozen(obj)); // Output: true
```

---

### **Object.defineProperty()**

**Description**: Defines a new property directly on an object.

```js
const obj = {};
Object.defineProperty(obj, 'name', {
  value: 'Hamza',
  writable: false,
});

console.log(obj.name); // Output: Hamza
obj.name = 'Ali'; // This will fail in strict mode
```

---

### **Object.defineProperties()**

**Description**: Defines multiple new properties on an object.

```js
const obj = {};
Object.defineProperties(obj, {
  name: { value: 'Hamza', writable: false },
  age: { value: 25, writable: true },
});

console.log(obj);
// Output: { name: 'Hamza', age: 25 }
```

---

### **Object.getOwnPropertyDescriptor()**

**Description**: Returns the property descriptor for a specific property.

```js
const obj = { name: 'Hamza' };

const descriptor = Object.getOwnPropertyDescriptor(obj, 'name');

console.log(descriptor);
// Output: { value: 'Hamza', writable: true, enumerable: true, configurable: true }
```

---

### **Object.getOwnPropertyDescriptors()**

**Description**: Returns all property descriptors of an object.

```js
const obj = { name: 'Hamza', age: 25 };

const descriptors = Object.getOwnPropertyDescriptors(obj);

console.log(descriptors);
/*
Output:
{
  name: { value: 'Hamza', writable: true, enumerable: true, configurable: true },
  age: { value: 25, writable: true, enumerable: true, configurable: true }
}
*/
```

---

### **Object.getOwnPropertyNames()**

**Description**: Returns an array of all properties (including non-enumerable properties).

```js
const obj = { name: 'Hamza', age: 25 };

console.log(Object.getOwnPropertyNames(obj));
// Output: ['name', 'age']
```

---

### **Object.getPrototypeOf()**

**Description**: Returns the prototype of an object.

```js
const obj = {};

console.log(Object.getPrototypeOf(obj));
// Output: {}
```

---

### **Object.is()**

**Description**: Compares two values for strict equality, like `===`, but handles special cases (e.g., `NaN`).

```js
console.log(Object.is(NaN, NaN)); // Output: true
console.log(Object.is(0, -0));   // Output: false
```

---

### **Object.hasOwn()**

**Description**: Checks if an object has a specific property as its own (not inherited).

```js
const obj = { name: 'Hamza' };

console.log(Object.hasOwn(obj, 'name')); // Output: true
console.log(Object.hasOwn(obj, 'age'));  // Output: false
```

---

### **Object.toString()**

**Description**: Returns a string representation of an object.

```js
const obj = {};

console.log(obj.toString());
// Output: [object Object]
```
