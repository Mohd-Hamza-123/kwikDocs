### **1. `pop()`**

Removes the last element from an array and returns it.

```javascript
let numbers = [1, 2, 3, 4];
let last = numbers.pop();
console.log(numbers); // Output: [1, 2, 3]
console.log(last);    // Output: 4
```

### **2. `push()`**

Adds one or more elements to the end of an array and returns the new length.

```javascript
let numbers = [1, 2, 3];
let newLength = numbers.push(4, 5);
console.log(numbers);  // Output: [1, 2, 3, 4, 5]
console.log(newLength); // Output: 5
```

### **3. `shift()`**

Removes the first element from an array and returns it.

```javascript
let numbers = [1, 2, 3, 4];
let first = numbers.shift();
console.log(numbers); // Output: [2, 3, 4]
console.log(first);   // Output: 1
```

### **4. `unshift()`**

Adds one or more elements to the beginning of an array and returns the new length.

```javascript
let numbers = [2, 3, 4];
let newLength = numbers.unshift(0, 1);
console.log(numbers);  // Output: [0, 1, 2, 3, 4]
console.log(newLength); // Output: 5
```

### **5. `concat()`**

Merges two or more arrays into a new array.

```javascript
let arr1 = [1, 2];
let arr2 = [3, 4];
let merged = arr1.concat(arr2, [5, 6]);
console.log(merged);  // Output: [1, 2, 3, 4, 5, 6]
console.log(arr1);    // Output: [1, 2] (original array is unchanged)
```

### **6. `indexOf()`**

Returns the index of the first occurrence of a value, or `-1` if not found.

```javascript
let numbers = [1, 2, 3, 4, 3];
console.log(numbers.indexOf(3)); // Output: 2
console.log(numbers.indexOf(5)); // Output: -1
```

### **7. `includes()`**

Checks if an array contains a specific value. Returns `true` or `false`.

```javascript
let fruits = ["apple", "banana", "mango"];
console.log(fruits.includes("banana")); // Output: true
console.log(fruits.includes("grape"));  // Output: false
```

### **8. `reverse()`**

Reverses the order of elements in an array.

```javascript
let numbers = [1, 2, 3, 4];
numbers.reverse();
console.log(numbers); // Output: [4, 3, 2, 1]
```

### **9. `slice()`**

Extracts a section of an array and returns a new array. Does not modify the original array.

```javascript
let numbers = [1, 2, 3, 4, 5];
let sliced = numbers.slice(1, 4); // From index 1 to index 4 (not inclusive)
console.log(sliced);  // Output: [2, 3, 4]
console.log(numbers); // Output: [1, 2, 3, 4, 5] (original remains unchanged)
```

### **10. `splice()`**

Adds or removes elements from an array.

#### Remove:

```javascript
let numbers = [1, 2, 3, 4, 5];
let removed = numbers.splice(1, 2); // Remove 2 elements starting at index 1
console.log(numbers); // Output: [1, 4, 5]
console.log(removed); // Output: [2, 3]
```

#### Add:

```javascript
let numbers = [1, 4, 5];
numbers.splice(1, 0, 2, 3); // Add 2, 3 at index 1
console.log(numbers); // Output: [1, 2, 3, 4, 5]
```

### **11. `forEach()`**

The `forEach()` method in JavaScript is used to execute a given function for each element in an array. Unlike other iteration methods, it doesn’t return a new array—it only performs side effects.

#### Syntax:
```javascript
array.forEach(callback(currentValue, index, array), thisArg)
```

#### Parameters:
- `callback`: A function executed for each element. It accepts the following arguments:
  - `currentValue`: The current element being processed.
  - `index` (optional): The index of the current element.
  - `array` (optional): The array `forEach` was called on.
- `thisArg` (optional): A value to use as `this` when executing the callback.

#### Example:
```javascript
const numbers = [1, 2, 3, 4];

numbers.forEach((number, index) => {
  console.log(`Index: ${index}, Value: ${number}`);
});
// Output:
// Index: 0, Value: 1
// Index: 1, Value: 2
// Index: 2, Value: 3
// Index: 3, Value: 4
```

#### Key Points:
- It modifies the original array if the callback makes changes.
- It doesn’t return anything (undefined).
- Best used for performing actions (like logging or updating external variables) rather than transformations.

### **12. `map()`**

The `map()` method creates a new array populated with the results of calling a provided function on every element in the calling array.

#### Syntax:
```javascript
array.map(callback(currentValue, index, array), thisArg)
```

#### Parameters:
- `callback`: A function executed for each element. It accepts the following arguments:
  - `currentValue`: The current element being processed.
  - `index` (optional): The index of the current element.
  - `array` (optional): The array `map` was called on.
- `thisArg` (optional): A value to use as `this` when executing the callback.

#### Example:
```javascript
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(number => number * 2);

console.log(doubled); // Output: [2, 4, 6, 8]
```

#### Key Points:
- It doesn’t modify the original array.
- It always returns a new array of the same length.
- Best used for transforming data.

### **13. `filter()`**

The `filter()` method creates a new array containing all elements of the calling array that pass a test implemented by the provided function.

#### Syntax:
```javascript
array.filter(callback(element, index, array), thisArg)
```

#### Parameters:
- `callback`: A function to test each element. It accepts the following arguments:
  - `element`: The current element being processed.
  - `index` (optional): The index of the current element.
  - `array` (optional): The array `filter` was called on.
- `thisArg` (optional): A value to use as `this` when executing the callback.

#### Example:
```javascript
const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter(number => number % 2 === 0);

console.log(evenNumbers); // Output: [2, 4]
```

#### Key Points:
- It doesn’t modify the original array.
- It returns a new array that can have fewer elements than the original.
- Best used for selecting a subset of data based on a condition.
