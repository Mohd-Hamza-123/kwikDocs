_Defination : **strings** are a data type used to represent and work with sequences of characters (text)._
----------------------------------------------------------------------------------------------------------

Property :

    const username = "mohd hamza"
    
    const charlength = username.length
    
    console.log(charlength) // output : 10

_Methods that takes no argument :_ 

_1) toUppercase  :_

    let text1 = "Hello World!";
    let text2 = text1.toUpperCase();
    console.log(text2); // HELLO WORLD!

_2) toLowerCase  :_

    let text1 = "Mohd Hamza";      
    let text2 = text1.toLowerCase();
    
    console.log(text2) // mohd hamza

_3) trim  :_

    let text1 = "          Mohd Hamza        ";      
    let text2 = text1.trim();
    
    console.log(text2) // Mohd Hamza

_4) trimStart  :_

    let text1 = "          Mohd Hamza        ";      
    let text2 = text1.trimStart();
    
    console.log(text2) // 'Mohd Hamza        '

_5) trimEnd  :_

    let text1 = "          Mohd Hamza        ";      
    let text2 = text1.trimEnd();
    
    console.log(text2) //     '          Mohd Hamza'

_Methods that takes  argument :_ 

_1) includes( )_ 

    let message = "I am Mohd Hamza. This is my website."
    
    let really = message.includes("This is my website")
    
    console.log(really) // true

    let message = "I am Mohd Hamza. This is my website."
    
    const result = message.includes("Mohd Khalid")
    
    console.log(result) // false

_2) indexOf ( )_

    let message = "I am Mohd Hamza.This is my website."
    
    const index1 = message.indexOf("M")
    const index2 = message.indexOf("Hamza")
    const index3 = message.indexOf("My Website")
    
    console.log(index1) // 5
    console.log(index2) // 10
    console.log(index3) // -1
    

3) replace( )  
  

    const text = "apple banana apple";
    const result = text.replace("apple", "orange");
    console.log(result); // Output: "orange banana apple"
    

4) replaceAll( )

    const text = "apple banana apple";
    const result = text.replaceAll("apple", "orange");
    console.log(result); // Output: "orange banana orange"
    

5) concat( )

    const str1 = "Hello";
    const str2 = "World";
    const result = str1.concat(" ", str2);
    console.log(result); // Output: "Hello World"
    

    const str1 = "I";
    const str2 = "love";
    const str3 = "JavaScript!";
    const result = str1.concat(" ", str2, " ", str3);
    console.log(result); // Output: "I love JavaScript!"
    

**While `concat()` works well, using the `+` operator is often simpler and more common :**

    const str1 = "Hello";
    const str2 = "World";
    const result = str1 + " " + str2;
    console.log(result); // Output: "Hello World"
    

6) padStart( )

    const invoiceNumber = "45";
    
    const result = invoiceNumber.padStart(6,'0');
    
    console.log(result) // Output: "000045"

7) padEnd( )

    const lastFourDigit = "9393";
    
    const hiddenAccountNo = lastFourDigit.padStart(16,"*")
    
    console.log(hiddenAccountNo) // output : ************9393

8) charAt( )

    const str = "Hello, World!";
    console.log(str.charAt(0)); // Output: "H"
    console.log(str.charAt(7)); // Output: "W"
    console.log(str.charAt(20)); // Output: "" (empty string if index is out of bounds)
    

9) charCodeAt( )

    const str = "Hello, World!";
    console.log(str.charCodeAt(0)); // Output: 72 (Unicode value of 'H')
    console.log(str.charCodeAt(7)); // Output: 87 (Unicode value of 'W')
    console.log(str.charCodeAt(20)); // Output: NaN (if index is out of bounds)
    

**`Note : charCodeAt()`** method gives the Unicode code point of a character at a specified index, which is often equivalent to the ASCII code for standard English characters.
