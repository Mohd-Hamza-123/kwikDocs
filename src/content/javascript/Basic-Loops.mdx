**1) For Loop :** 

    for (initialization; condition; increment/decrement) {
        // Code to execute in each iteration
    }
    

Eg : 
```js
    // Print numbers from 1 to 5 
    for (let i = 1; i <= 5; i++) {
        console.log(i);
    }
    
    // 1
    // 2
    // 3
    // 4
    // 5
    ```

_Break and Continue :_
```js
    for (let i = 1; i <= 10 ; i++) {
        if (i === 3) {
            continue; // Skip the loop when i is 3
        }
        if (i === 5) {
            break; // Exit the loop when i is 5
        }
        console.log(i);
    }
```
