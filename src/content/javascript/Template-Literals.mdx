    let text = `Hello World!`;

    let text =
    `The quick

*   **Template String** provide an easy way to interpolate variables and expressions into strings.Â The method is called string interpolation.

    let firstName = "John";
    let lastName = "Doe";
    
    let text = `Welcome ${firstName}, ${lastName}!`;
    console.log(text); // Outputs: Welcome John, Doe!
