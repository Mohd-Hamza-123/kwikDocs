### Definition of Strings

Strings are a data type used to represent and work with sequences of characters (text).

---

### Property of Strings

```js
const username = "mohd hamza";
const charLength = username.length;
console.log(charLength); // Output: 10
```

---

### Methods That Take No Arguments

#### 1. `toUpperCase()`
Converts a string to uppercase.

```js
let text1 = "Hello World!";
let text2 = text1.toUpperCase();
console.log(text2); // Output: "HELLO WORLD!"
```

#### 2. `toLowerCase()`
Converts a string to lowercase.

```js
let text1 = "Mohd Hamza";
let text2 = text1.toLowerCase();
console.log(text2); // Output: "mohd hamza"
```

#### 3. `trim()`
Removes whitespace from both ends of a string.

```js
let text1 = "          Mohd Hamza        ";
let text2 = text1.trim();
console.log(text2); // Output: "Mohd Hamza"
```

#### 4. `trimStart()`
Removes whitespace from the start of a string.

```js
let text1 = "          Mohd Hamza        ";
let text2 = text1.trimStart();
console.log(text2); // Output: "Mohd Hamza        "
```

#### 5. `trimEnd()`
Removes whitespace from the end of a string.

```js
let text1 = "          Mohd Hamza        ";
let text2 = text1.trimEnd();
console.log(text2); // Output: "          Mohd Hamza"
```

---

### Methods That Take Arguments

#### 1. `includes()`
Checks if a string contains a specified substring.

```js
let message = "I am Mohd Hamza. This is my website.";
let result = message.includes("This is my website");
console.log(result); // Output: true

result = message.includes("Mohd Khalid");
console.log(result); // Output: false
```

#### 2. `indexOf()`
Finds the index of the first occurrence of a substring.

```js
let message = "I am Mohd Hamza. This is my website.";
console.log(message.indexOf("M"));       // Output: 5
console.log(message.indexOf("Hamza"));   // Output: 10
console.log(message.indexOf("My Website")); // Output: -1
```

#### 3. `replace()`
Replaces the first occurrence of a specified value.

```js
const text = "apple banana apple";
const result = text.replace("apple", "orange");
console.log(result); // Output: "orange banana apple"
```

#### 4. `replaceAll()`
Replaces all occurrences of a specified value.

```js
const text = "apple banana apple";
const result = text.replaceAll("apple", "orange");
console.log(result); // Output: "orange banana orange"
```

#### 5. `concat()`
Concatenates two or more strings.

```js
const str1 = "Hello";
const str2 = "World";
const result = str1.concat(" ", str2);
console.log(result); // Output: "Hello World"

const str3 = "I";
const str4 = "love";
const str5 = "JavaScript!";
console.log(str3.concat(" ", str4, " ", str5)); // Output: "I love JavaScript!"
```

**Alternative Using `+` Operator:**

```js
const str1 = "Hello";
const str2 = "World";
console.log(str1 + " " + str2); // Output: "Hello World"
```

#### 6. `padStart()`
Pads the start of a string with a specified character until it reaches a specified length.

```js
const invoiceNumber = "45";
console.log(invoiceNumber.padStart(6, '0')); // Output: "000045"
```

#### 7. `padEnd()`
Pads the end of a string with a specified character until it reaches a specified length.

```js
const lastFourDigits = "9393";
const hiddenAccountNo = lastFourDigits.padStart(16, "*");
console.log(hiddenAccountNo); // Output: "************9393"
```

#### 8. `charAt()`
Returns the character at a specified index.

```js
const str = "Hello, World!";
console.log(str.charAt(0));  // Output: "H"
console.log(str.charAt(7));  // Output: "W"
console.log(str.charAt(20)); // Output: "" (empty string)
```

#### 9. `charCodeAt()`
Returns the Unicode value of the character at a specified index.

```js
const str = "Hello, World!";
console.log(str.charCodeAt(0));  // Output: 72 (Unicode value of 'H')
console.log(str.charCodeAt(7));  // Output: 87 (Unicode value of 'W')
console.log(str.charCodeAt(20)); // Output: NaN (out of bounds)
```

> **Note**: `charCodeAt()` provides the Unicode code point of a character, often equivalent to the ASCII code for standard English characters.
