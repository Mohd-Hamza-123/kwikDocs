### Methods on Math Object

#### 1. Math.abs(x)

Returns the absolute value of a number.

```js
console.log(Math.abs(-5));    // 5
console.log(Math.abs(-4.4));  // 4.4
console.log(Math.abs(0));     // 0
console.log(Math.abs(-0.1));  // 0.1
console.log(Math.abs(3));     // 3
```

#### 2. Math.round(x)

Rounds a number to the nearest integer.

```js
console.log(Math.round(2.2)); // 2
console.log(Math.round(3.9)); // 4
console.log(Math.round(5.1)); // 5
console.log(Math.round(-1.1)); // -1
console.log(Math.round(-1.5)); // -1
```

#### 3. Math.ceil(x)

Rounds a number up to the nearest integer.

```js
console.log(Math.ceil(4.2)); // 5
console.log(Math.ceil(-4.2)); // -4
```

#### 4. Math.floor(x)

Rounds a number down to the nearest integer.

```js
console.log(Math.floor(5.95)); // 5
console.log(Math.floor(5.05)); // 5
console.log(Math.floor(5));    // 5
console.log(Math.floor(-5.05)); // -6
```

#### 5. Math.sqrt(x)

Calculates the square root of a number.

```js
console.log(Math.sqrt(2)); // 1.4142135623730951
```

#### 6. Math.pow(x, y)

Raises `x` to the power of `y`.

```js
console.log(Math.pow(4, 2)); // 16
```

#### 7. Math.random()

Generates a floating-point number between **0 (inclusive)** and **1 (exclusive)**.

```js
console.log(Math.random());
```
