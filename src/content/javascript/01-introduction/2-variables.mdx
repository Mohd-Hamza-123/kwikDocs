### **Variables** 

Variables are Containers for Storing Data. All JavaScript variables must be identified with unique names. These unique names are called identifiers.


<ImageCenter src="https://res.cloudinary.com/dkkifg17t/image/upload/v1736609157/Documentarium/JavaScript/bf6igotrocf36ded0cmg.jpg" alt="Variables" />

JavaScript Variables can be declared in 4 ways:

* Automatically
* Using `var`
* Using `let`
* Using `const`

#### **Automatically**

```javascript
x = 5;
y = 6;
z = x + y;
```
#### **using `var`**

```javascript
var x = 5;
var y = 6;
var z = x + y;
```

#### **using `let`**

```javascript
let x = 5;
let y = 6;
let z = x + y;
```

#### **using `const`**

```javascript
const x = 5;
const y = 6;
const z = x + y;
```


**Rules for naming variables in JavaScript:**

*   Variable names must start with a letter, an underscore (`_`) or a dollar sign (`$`).
*   Variable names cannot contain spaces or any special characters ( % , ^ , &  etc ).
*   Variables cannot be the same as reserved keywords such as `if` or `const`.
*   By convention, JavaScript variable names are written in camelCase.
*   Variables should be given descriptive names that indicate their content and usage (e.g. `sellingPrice` and `costPrice` rather than `x` and `y`).
