Functions in JavaScript are reusable blocks of code that perform a specific task.

Function Declaration : 

    function functionName(parameters) {
        // code to execute
        return value; // optional
    }
    

Eg : 

    function greet() {
        console.log("Hello!");
    }
    greet(); // Function is called by its name
    

Function Expression  : 

A **function expression** is a function that is assigned to a variable. When you store a function in a variable, it becomes a function expression. 

    const variableName = function(parameters) {
      // Function body
    };
    

Eg: 

    // Anonymous Function Expression
    const greet = function(name) {
      return `Hello, ${name}!`;
    };
    
    // Named Function Expression
    const greet = function greeting(name) {
      return `Hello, ${name}!`;
    };
    

Making our own math.floor() method :  

    const math = {
        floor: function (num) {
            if (num < 0) {
                return parseInt(num) - 1
            } else if (num > 0) {
                return parseInt(num);
            } else {
                return num
            }
        }
    }
    
    const x = Math.floor(5.5)
    const y = Math.floor(-5.6)
    console.log(x);
    console.log(y);
    
    const a = math.floor(5.5)
    const b = math.floor(-5.6)
    console.log(a)
    console.log(b)
