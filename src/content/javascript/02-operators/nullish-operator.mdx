The `??` operator returns the **right-hand value only if the left-hand value is `null` or `undefined`**.

It is used to provide **default values** for variables that might be `null` or `undefined`, without mistakenly treating values like `0`, `""` (empty string), or `false` as missing.


If `value` is `null` or `undefined`, `result` will be `defaultValue`.
Otherwise, `result` will be `value`.
