The ternary operator is a shorthand for `if...else`.
It evaluates a condition and returns one of two values based on whether the condition is `true` or `false`.

---

### ‚úÖ Syntax:

```js
condition ? value_if_true : value_if_false;
```

---

### üî∏ Example 1: Basic usage

```js
let isVerified = true;

let message = isVerified ? "Access granted" : "Access denied";

console.log(message); // Output: Access granted
```

üìù If `isVerified` is true ‚Üí `message` gets `"Access granted"`
If false ‚Üí `message` would be `"Access denied"`

---

### üî∏ Example 2: Checking age

```js
let age = 17;

let access = age >= 18 ? "Allowed" : "Not allowed";

console.log(access); // Output: Not allowed
```

---

### üî∏ Example 3: Nested ternary (not recommended for readability)

```js
let score = 75;

let grade = score >= 90 ? "A" :
            score >= 80 ? "B" :
            score >= 70 ? "C" : "F";

console.log(grade); // Output: C
```

> ‚ö†Ô∏è Use nested ternary carefully ‚Äî it's short but can become hard to read.

---

### ‚úÖ Real Use: Inline in UI (like React)

```js
let isLoggedIn = false;

let buttonText = isLoggedIn ? "Logout" : "Login";

console.log(buttonText); // Output: Login
```
