JavaScript does not have many "string-specific" operators, but it allows you to work with strings using:

1. **Concatenation operator (`+`)**
2. **Concatenation assignment operator (`+=`)**
3. **Other methods** (not operators, like `.includes()`, `.slice()`, etc.)

We'll focus here on the actual **operators** that involve strings.

---

### ‚úÖ 1. **`+` (Concatenation Operator)**

#### **Definition:**

Combines (joins) two or more strings into one.

#### **Example:**

```js
let firstName = "Hamza";
let lastName = "Ali";

let fullName = firstName + " " + lastName;

console.log(fullName); // Output: Hamza Ali
```

üìù You can also concatenate strings with numbers (numbers are converted to strings):

```js
let age = 25;
let message = "I am " + age + " years old.";

console.log(message); // Output: I am 25 years old.
```

---

### ‚úÖ 2. **`+=` (Concatenation Assignment Operator)**

#### **Definition:**

Appends the right-hand string to the left-hand string and updates the variable.

#### **Example:**

```js
let text = "Hello";
text += " World";

console.log(text); // Output: Hello World
```

---

### ‚ùó Be Careful: With `+`, order matters

```js
let result = "The value is: " + 10 + 5;
console.log(result); // Output: The value is: 105

let result2 = 10 + 5 + " is the value";
console.log(result2); // Output: 15 is the value
```

üìù Left-to-right evaluation: JavaScript converts numbers to strings if it encounters a string first.

---

### üõ† Summary

| Operator | Description              | Example                  | Result       |
| -------- | ------------------------ | ------------------------ | ------------ |
| `+`      | Concatenates strings     | `"Hi " + "there"`        | `"Hi there"` |
| `+=`     | Appends string to itself | `msg = "Hi"; msg += "!"` | `"Hi!"`      |
