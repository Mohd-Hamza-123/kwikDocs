### Addition `+`

The addition operator adds two numbers together.

```js
let a = 5;
let b = 3;
let sum = a + b; // Sum of 5 and 3
console.log(sum); // Output: 8
```

### Subtraction `-`

The subtraction operator subtracts the second number from the first.

```js
let a = 5;
let b = 3;
let difference = a - b; // Difference between 5 and 3
console.log(difference); // Output: 2
```

### Multiplication `*`

The multiplication operator multiplies two numbers together.

```js
let a = 5;
let b = 3;
let product = a * b; // Product of 5 and 3
console.log(product); // Output: 15
```

### Division `/`

The division operator divides the first number by the second.

```js
let a = 10;
let b = 3;
let quotient = a / b; // Quotient when 10 is divided by 3
console.log(quotient); // Output: 3.3333333333333335
```

### Modulus (Remainder) `%`

The modulus operator returns the remainder of the division of two numbers.

```js
let a = 10;
let b = 3;
let remainder = a % b; // Remainder when 10 is divided by 3
console.log(remainder); // Output: 1
```

### Exponentiation `**`

The exponentiation operator raises the base to the power of the exponent.

```js
let base = 2;
let exponent = 3;
let result = base ** exponent; // 2 raised to the power of 3
console.log(result); // Output: 8
```

### Increment (`++`)

The increment operator increases the value of a variable by 1.

#### Post-Increment

The value is returned **before** incrementing.

```js
let a = 5;
console.log(a++); // Output: 5 (returns value before increment)
console.log(a);   // Output: 6 (value after increment)
```

#### Pre-Increment

The value is incremented **before** being returned.

```js
let b = 5;
console.log(++b); // Output: 6 (increments first, then returns)
console.log(b);   // Output: 6
```

### Decrement (`--`)

The decrement operator decreases the value of a variable by 1.

#### Post-Decrement

The value is returned **before** decrementing.

```js
let x = 5;
console.log(x--); // Output: 5 (returns value before decrement)
console.log(x);   // Output: 4 (value after decrement)
```

#### Pre-Decrement

The value is decremented **before** being returned.

```js
let y = 5;
console.log(--y); // Output: 4 (decrements first, then returns)
console.log(y);   // Output: 4
```

> **Note**: Increment and decrement operators (`++` and `--`) can only be used on variables, not on constants or literal values.
