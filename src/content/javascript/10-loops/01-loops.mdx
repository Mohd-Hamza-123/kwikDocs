In JavaScript, **loops** are used to repeatedly execute a block of code as long as a specified condition is true. Different types of loops serve different purposes.

### **1. For Loop**

The `for` loop repeats a block of code a certain number of times.

```javascript
for (initialization; condition; increment/decrement) {
    // Code to execute in each iteration
}
```

#### Example:

```javascript
// Print numbers from 1 to 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

// Output:
// 1
// 2
// 3
// 4
// 5
```

#### Break and Continue:

```javascript
for (let i = 1; i <= 10; i++) {
    if (i === 3) {
        continue; // Skip the loop iteration when i is 3
    }
    if (i === 5) {
        break; // Exit the loop when i is 5
    }
    console.log(i);
}

// Output:
// 1
// 2
// 4
```

### **2. While Loop**

The `while` loop executes a block of code as long as the specified condition is true.

```javascript
while (condition) {
    // Code to execute
}
```

#### Example:

```javascript
// Print numbers from 1 to 5
let i = 1;
while (i <= 5) {
    console.log(i);
    i++;
}

// Output:
// 1
// 2
// 3
// 4
// 5
```

### **3. Do-While Loop**

The `do-while` loop is similar to the `while` loop, but it executes the block of code at least once, regardless of the condition.

```javascript
do {
    // Code to execute
} while (condition);
```

#### Example:

```javascript
// Print numbers from 1 to 5
let i = 1;
do {
    console.log(i);
    i++;
} while (i <= 5);

// Output:
// 1
// 2
// 3
// 4
// 5
```

### **4. For...of Loop**

The `for...of` loop iterates over iterable objects such as arrays, strings, or sets.

```javascript
for (element of iterable) {
    // Code to execute for each element
}
```

#### Example:

```javascript
const fruits = ["Apple", "Banana", "Cherry"];
for (const fruit of fruits) {
    console.log(fruit);
}

// Output:
// Apple
// Banana
// Cherry
```

### **5. For...in Loop**

The `for...in` loop iterates over the enumerable properties of an object.

```javascript
for (key in object) {
    // Code to execute for each property
}
```

#### Example:

```javascript
const person = { name: "John", age: 30, city: "New York" };
for (const key in person) {
    console.log(`${key}: ${person[key]}`);
}

// Output:
// name: John
// age: 30
// city: New York
