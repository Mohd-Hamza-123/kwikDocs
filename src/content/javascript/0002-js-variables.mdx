### **Variables :** _Variables are Containers for Storing Data._  _All JavaScript variables must be identified with unique names. These unique names are called identifiers._

_![](https://res.cloudinary.com/dkkifg17t/image/upload/v1736609157/Documentarium/JavaScript/bf6igotrocf36ded0cmg.jpg)_

JavaScript Variables can be declared in 4 ways:

*   Automatically
*   Using `var`
*   Using `let`
*   Using `const`

Automatically

```js
    x = 5;
    y = 6;
    z = x + y;
```
Using `var`

```js
    var x = 5;
    var y = 6;
    var z = x + y;
```

Using `let`

```js
    let x = 5;
    let y = 6;
    let z = x + y;
```
Using `const`

```js
    const x = 5;
    const y = 6;
    const z = x + y;
```

Mixed Example
```js
    const price1 = 5;
    const price2 = 6;
    let total = price1 + price2;
```

_**Rules for naming variables :**_

*   Variable names must start with a letter, an underscore (`_`) or a dollar sign (`$`).
*   Variable names cannot contain spaces or any special characters ( % , ^ , &  etc ).
*   Variables cannot be the same as reserved keywords such as `if` or `const`.
*   By convention, JavaScript variable names are written in camelCase.
*   Variables should be given descriptive names that indicate their content and usage (e.g. `sellingPrice` and `costPrice` rather than `x` and `y`).
