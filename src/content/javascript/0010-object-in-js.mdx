An object is **a collection of properties.  P****roperties** are the key-value pairs inside the object. 

*   **Keys**: These are the names (or identifiers) of the properties.
*   **Values**: These are the data or functions assigned to the keys

```js
    // Syntax 
    
    let objectName = {
      key1: value1,
      key2: value2,
      key3: value3
    };
 ```   

**1) Creating Object**
```js
    // Empty Object
    
    const obj1 = {}

    const obj2 = {
        name: "Mohammad Hamza",
        age: 7,
        email: "hamza@gmail.com",
        isMale: true,
        phoneNo: 393928282
    }
```
**2) Updating Object**
```js
    let person = {
      name: "Hamza",
      age: 14
    };
    
    // Update a property
    person.age = 50;
    
    console.log(person); 
    // Output: { name: "Hamza", age: 50 }
    
    let car = {
      brand: "Tesla",
      "model name": "Model S"
    };
    
    // Update a property
    car["model name"] = "Model X";
    
    console.log(car);
    // Output: { brand: "Tesla", "model name": "Model X" }
    ```

**3) Adding new Property**
```js
    let user = {
      name: "Ali"
    };
    
    // Add a new property
    user.email = "ali@example.com";
    
    console.log(user);
    // Output: { name: "Ali", email: "ali@example.com" }
    ```

**4) Deleting Properties**
```js
    let person = {
      name: "Hamza",
      age: 25,
      city: "New York"
    };
    
    // Delete a property using dot notation
    delete person.age;
    
    // Delete a property using bracket notation
    delete person["city"];
    
    console.log(person);
    // Output: { name: "Hamza" }
    ```

**Nested Objects** 
```js
    // Nested Objects
    
    const myDetails = {
        name: {
            firstName: "Mohammad",
            lastName: "Hamza"
        },
        age: 7,
        email: "hamza@gmail.com",
        isMale: true,
        address: {
            city: "Lucknow",
            pincode: 38383,
            state: "UP",
            cityDetails: {
                population: "2 crore",
                language: "Hindi",
                famousFood: "Chicken Biryani"
            }
        }
    }
```
Each object (nested or otherwise) is allocated separately in memory, and the parent object holds references to its child objects.

**Object using symbols** 
```js
    const sym1 = Symbol("key1");
    
    const obj = {
        [sym1] : "Mohd Hamza"
    }
    
    console.log(obj[sym1]);
```
**Using this in Objects** 
```js
    // using this in objects
    
    const myDetails = {
        name: "Mohd Hamza",
        country: "India",
        info: function getInfo() {
            console.log(`My name is ${this.name} & I live in ${this.country}`);
        }
    }
    
    console.log(myDetails.info())
```
