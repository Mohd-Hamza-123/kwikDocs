Nodemon is a tool that automatically restarts your Node.js application when you make changes to the source code.

To install Nodemon, run the following command in your terminal:

```bash
npm install -D nodemon
```

To use Nodemon, open your terminal and navigate to the directory where your Node.js application is located. Then, run the following command:

```bash
nodemon app.js
```
