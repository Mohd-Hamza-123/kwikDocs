### Syntax of `fs.writeFile()`

```javascript
fs.writeFile(path, data[, options], callback);
```

### Parameters

* **path**: The path to the file to create.
* **data**: The data to write to the file.
* **options**: An object with optional settings.
* **callback**: A callback function that is called when the file is written.

### Options

* **encoding**: The encoding to use for the file content. Default is `'utf8'`.
* **mode**: The mode to use for the file. Default is `0o666`.

Example : 

```js
import http from 'http';
import path from 'path';
import fs from 'fs';

const server = http.createServer((req, res) => {
    const filePath = path.join('public', 'sample.txt');
    const content = 'Hello World!';
    fs.writeFile(filePath, content, (err) => {
        if (err) {
            res.writeHead(500, { 'Content-Type': 'text/plain' });
            res.end("Error: Can't create the file");
            return;
        }
        res.writeHead(200, { 'Content-Type': 'text/plain' });
        res.end("File created successfully!");
    });
});

server.listen(8000)
```

<Callout>Updating file also works in the same way as creating a file.</Callout>
