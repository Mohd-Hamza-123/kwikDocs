#### Syntax of `fs.readFile()`

```javascript
fs.readFile(path[, options], callback);
```

### Parameters

* **path**: The path to the file to read.
* **options**: An object with optional settings.
* **callback**: A callback function that is called when the file is read.

### Options

* **encoding**: The encoding to use for the file content. Default is `'utf8'`.
* **flag**: The string denoting the mode in which the file is opened. Default is `'r'`.


Example : 

```javascript
import http from 'http';
import path from 'path';
import fs from 'fs';

const server = http.createServer((req, res) => {
    fs.readFile(path.join('public', 'sample.txt'), 'utf8', (err, data) => {
        if (err) throw new Error("Can't Read File");
        res.end(data)
    })
});

server.listen(8000)
```
