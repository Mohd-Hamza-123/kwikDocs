### Syntax of `fs.promises.readFile()`

```javascript
fs.promises.readFile(path, options);
```

* **`path`**: Path to the file (string, Buffer, or URL).
* **`options`**: (Optional) Object or string specifying:

  * `encoding`: File encoding (e.g., `'utf8'`).
  * `flag`: File system flag (`'r'` by default).

**Returns:** A Promise that resolves with the file's content or rejects with an error.

---

### Example: Read a File

This example reads `sample.txt` from the `public` directory:

```javascript
import fs from 'fs/promises';
import path from 'path';

const readFile = async () => {
    try {
        const filePath = path.join('public', 'sample.txt');
        const data = await fs.readFile(filePath, 'utf8');
        console.log('File content:', data);
    } catch (error) {
        console.error('Error reading file:', error);
    }
};

readFile();
```

---

### Notes

* **Platform-Independent Paths:** Use `path.join()` to avoid issues with file paths.
* **Error Handling:** Use `try-catch` to handle errors like missing files or permission issues.
* **Readable Code:** `async/await` ensures clean and manageable file operations.
