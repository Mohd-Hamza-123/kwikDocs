### Syntax of `fs.promises.access()`

```javascript
fs.promises.access(path, mode);
```

#### Parameters:

* **`path`**: Path to the file (string, Buffer, or URL).
* **`mode`**: (Optional) Integer specifying the type of access check. Defaults to checking for the fileâ€™s existence (`fs.constants.F_OK`).

**Returns:** A Promise that resolves if the file exists and is accessible or rejects with an error.

---

### Example: Check File Existence

This example checks if a file named `sample.txt` exists in the `public` directory:

```javascript
import fs from 'fs/promises';
import path from 'path';

const checkFileExists = async () => {
    try {
        const filePath = path.join('public', 'sample.txt');
        await fs.access(filePath);
        console.log('File exists!');
    } catch (error) {
        console.log('File does not exist or is inaccessible.');
    }
};

checkFileExists();
```

---

### Notes

1. **Access Modes:**
   You can use access modes from `fs.constants` for specific checks:

   * `fs.constants.F_OK`: Check if the file exists (default).
   * `fs.constants.R_OK`: Check if the file is readable.
   * `fs.constants.W_OK`: Check if the file is writable.

2. **Error Handling:**

   * Use a `try-catch` block to handle errors for non-existent or inaccessible files.
