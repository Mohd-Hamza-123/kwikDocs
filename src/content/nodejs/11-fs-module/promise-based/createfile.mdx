### Syntax of `fs.promises.writeFile()`

```javascript
fs.promises.writeFile(path, data, options);
```

* **`path`**: Path to the file (string, Buffer, or URL).
* **`data`**: Data to be written to the file.
* **`options`**: (Optional) Object or string specifying:

  * `encoding`: File encoding (e.g., `'utf8'`).
  * `mode`: File mode (e.g., `0o666`).

**Returns:** A Promise that resolves with the number of bytes written or rejects with an error.

---

### Example: Create a File

This example creates a file named `sample.txt` in the `public` directory:

```javascript
import fs from 'fs/promises';
import path from 'path';

const createFile = async () => {
    try {
        const filePath = path.join('public', 'sample.txt');
        const content = 'Hello World!';
        await fs.writeFile(filePath, content, 'utf8');
        console.log('File created successfully!');
    } catch (error) {
        console.error('Error creating file:', error);
    }
};

createFile();
```
