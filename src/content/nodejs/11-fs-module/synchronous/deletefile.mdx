### Syntax of `fs.unlinkSync()`

```javascript
fs.unlinkSync(path);
```

Example : 

```js
import fs from 'fs';
import path from 'path';

console.log('deleting file');
try {
    fs.unlinkSync(path.join('public', 'sample.txt'));
    console.log('File deleted successfully!');
} catch (err) {
    console.error('Error deleting file:', err);
}
console.log('File deleted successfully!');
```
