### Syntax of `fs.readFileSync()`

```javascript
fs.readFileSync(path[, options]);
```

### Parameters

* **path**: The path to the file to read.
* **options**: An object with optional settings.

### Options

* **encoding**: The encoding to use for the file content. Default is `'utf8'`.
* **flag**: The string denoting the mode in which the file is opened. Default is `'r'`.

Example : 

```js
import fs from 'fs';
import path from 'path';

console.log('reading file');
try {
    const data = fs.readFileSync(path.join('public', 'sample.txt'), 'utf8');
    console.log(data);
} catch (err) {
    console.error('Error reading file:', err);
}
console.log('File read successfully');
```
