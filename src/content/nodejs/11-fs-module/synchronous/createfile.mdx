### Syntax of `fs.writeFileSync()`

```javascript
fs.writeFileSync(path, data, options);
```

### Parameters

* **path**: The path to the file to create.
* **data**: The data to write to the file.
* **options**: An object with optional settings.

### Options

* **encoding**: The encoding to use for the file content. Default is `'utf8'`.
* **mode**: The mode to use for the file. Default is `0o666`.

Example : 

```js
const filePath = path.join('public', 'writefile.txt');
const content = 'This is some example text to write into the file.';

try {
    console.log('Writing to file...');
    fs.writeFileSync(filePath, content, 'utf8');
    console.log('File written successfully!');
} catch (err) {
    console.error('Error writing to file:', err);
}
```
