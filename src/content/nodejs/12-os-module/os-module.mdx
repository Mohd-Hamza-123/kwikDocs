The `os` module is a core module in Node.js and does not require installation. Import it using:

```javascript
import os from 'os'; // ES6 Import
```

---

### Commonly Used Methods in the `os` Module

#### 1. **System Information**

* **`os.type()`**: Returns the operating system name.

  ```javascript
  console.log(os.type()); // Example: 'Linux', 'Darwin', 'Windows_NT'
  ```

* **`os.platform()`**: Returns the platform of the OS.

  ```javascript
  console.log(os.platform()); // Example: 'win32', 'darwin', 'linux'
  ```

* **`os.arch()`**: Returns the CPU architecture.

  ```javascript
  console.log(os.arch()); // Example: 'x64', 'arm'
  ```

* **`os.release()`**: Returns the operating system release version.

  ```javascript
  console.log(os.release()); // Example: '10.0.19043'
  ```

---

#### 2. **CPU and Memory Information**

* **`os.cpus()`**: Returns an array of objects containing CPU details.

  ```javascript
  console.log(os.cpus()); // Example: [{ model: 'Intel(R) Core(TM)...', speed: 2800, ... }]
  ```

* **`os.totalmem()`**: Returns the total system memory in bytes.

  ```javascript
  console.log(os.totalmem()); // Example: 17179869184 (16 GB)
  ```

* **`os.freemem()`**: Returns the available system memory in bytes.

  ```javascript
  console.log(os.freemem()); // Example: 8589934592 (8 GB)
  ```

---

#### 3. **Network Information**

* **`os.networkInterfaces()`**: Returns network interface details.

  ```javascript
  console.log(os.networkInterfaces());
  ```

---

#### 4. **User Information**

* **`os.userInfo([options])`**: Returns information about the current user.

  ```javascript
  console.log(os.userInfo()); // Example: { username: 'john', homedir: '/home/john', ... }
  ```

---

#### 5. **Directory Paths**

* **`os.homedir()`**: Returns the home directory of the current user.

  ```javascript
  console.log(os.homedir()); // Example: '/Users/john' or 'C:\\Users\\John'
  ```

* **`os.tmpdir()`**: Returns the system's temporary directory.

  ```javascript
  console.log(os.tmpdir()); // Example: '/tmp' or 'C:\\Users\\John\\AppData\\Local\\Temp'
  ```

---

#### 6. **System Uptime**

* **`os.uptime()`**: Returns the system uptime in seconds.

  ```javascript
  console.log(os.uptime()); // Example: 3600 (1 hour)
  ```

---

### Notes

* The `os` module is platform-independent, but some methods may return different results depending on the operating system.
* The data is often used to make OS-specific decisions or gather diagnostics.
