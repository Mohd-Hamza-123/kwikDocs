### Path Module Common Methods

The `path` module in Node.js is a built-in module that provides utilities for working with file and directory paths. It helps you handle file paths in a platform-independent way, making your code more portable and reliable.

---

### **Why Use the `path` Module?**

File paths differ across operating systems :

* **Windows** uses backslashes (`\`) for paths, e.g., `E:\folder\file.txt`.
* **Unix/Linux/macOS** uses forward slashes (`/`), e.g., `/home/user/file.txt`.

The `path` module abstracts these differences and provides consistent methods to work with paths, regardless of the platform.

---

### **Key Features**

1. **Platform Independence**:

   * Automatically handles platform-specific path separators (`\` or `/`).

2. **Path Manipulation**:

   * Resolve, normalize, join, or extract parts of file paths.

3. **Ease of Use**:

   * Simplifies working with relative and absolute paths.

---

### **Commonly Used Methods**

Here are some of the most useful methods in the `path` module:

#### **1. `path.join()`**

* Joins multiple path segments into a single path.
* Automatically adds the correct separator (`\` or `/`).

```javascript
import path from 'path';
const filePath = path.join('folder', 'subfolder', 'file.txt');
console.log(filePath); // Outputs: folder/subfolder/file.txt (on Unix) or folder\subfolder\file.txt (on Windows)
```

#### **2. `path.resolve()`**

* Takes multiple path parts (strings) as input
* Joins them together from `right to left`
* Returns an absolute path (full path from root)

```javascript
import path from 'path';

// Combine path segments into one absolute path
const fullPath = path.resolve('folder', 'subfolder', 'file.txt');
console.log(fullPath);
// On Unix: /current/working/dir/folder/subfolder/file.txt
// On Windows: C:\current\working\dir\folder\subfolder\file.txt
```

More examples: 

```javascript
import path from 'path';
console.log(path.resolve("folder","subfolder","file.txt"))
// E:\Javascript\Node JS\folder\subfolder\file.txt
console.log(path.resolve("/folder","subfolder","file.txt"))
// E:\folder\subfolder\file.txt
```

```javascript
import path from 'path';
console.log(path.resolve('folder', 'subfolder', '..', 'note.txt'))
// Output: E:\Javascript\Node JS\folder\note.txt
```

#### **3. `path.basename()`**

* Extracts the file name from a path.

```javascript
import path from 'path';
const filePath = '/folder/subfolder/file.txt';
console.log(path.basename(filePath)); // Outputs: file.txt
```

#### **4. `path.dirname()`**

* Extracts the directory part of a path.

```javascript
import path from 'path';
console.log(path.dirname('/folder/subfolder/file.txt')); // Outputs: /folder/subfolder
```

#### **5. `path.extname()`**

* Extracts the extension of a file.

```javascript
import path from 'path';
console.log(path.extname('file.txt')); // Outputs: .txt
```

#### **6. `path.normalize()`**

* Normalizes a path, resolving `..` and `.` segments.

```javascript
import path from 'path';
console.log(path.normalize('/folder//subfolder/../file.txt')); // Outputs: /folder/file.txt
```

#### **7. `path.isAbsolute()`**

* Checks if a path is absolute.

```javascript
import path from 'path';
console.log(path.isAbsolute('/folder/file.txt')); // true
console.log(path.isAbsolute('folder/file.txt'));  // false
```
