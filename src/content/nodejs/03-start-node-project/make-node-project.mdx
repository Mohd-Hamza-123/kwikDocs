1. Initialize a New Project

To create a Node.js project, you can use the following command:

```bash
npm init 
# or
npm init -y  # -y skips prompts and uses defaults
```

<Callout>
This will create a package.json file in your current directory.
</Callout>


2. Create Your Entry File
Create an index.js (or app.js) file.

Add a simple script:

```javascript
console.log("Hello, Node.js!");
```

4. Run Your Project

```bash
node index.js
```
