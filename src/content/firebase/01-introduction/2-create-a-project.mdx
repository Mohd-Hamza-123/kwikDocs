### 1) Create an Account on Firebase

After signing up, go to the **Firebase Console** and create a new project.

<ImageCenter src="https://res.cloudinary.com/dkkifg17t/image/upload/v1756915465/Screenshot_2025-09-03_170133_anc8xu.png" alt="Create a Project" />  

> Click on **Get Started** to begin creating a new project.

### 2) Enter Project Name

Provide a name for your project and continue.

### 3) Continue Setup

Click **Continue** to move to the next step.

### 4) Create Project

Once the setup is complete, your Firebase project will be created successfully.

---

### Create a Web App

* Now, let's create a web app. Click on the icon `</>`

<ImageCenter src="https://res.cloudinary.com/dkkifg17t/image/upload/v1756916439/Screenshot_2025-09-03_214623_yot3uf.jpg" alt="Create a Project" />

* Enter the name of your app and click on **Create** / **Register App** etc.

* You will get firebase setup like project id, app id, api key etc.

* Go to console.
