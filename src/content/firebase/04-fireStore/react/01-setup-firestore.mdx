### 1) To use FireStore Database click on `create Database` : 

<ImageCenter src="https://res.cloudinary.com/dkkifg17t/image/upload/v1757666715/images_1_oqaxki.png" alt="Setup Firestore" />


### 2) Choose `Standard Edition` : 


### 3) Enter Location : 

Choose a location for your database. Keep database id default or choose a new one.

Click `Create`

> Database will be created in a few minutes. 

### 3) Select Mode 

* use `Test Mode` for testing purpose.
* use `Production Mode` for production purpose.

> You can change the mode later.


Note : If you are using `Production Mode` then you need to change the rules to allow read and write access to your database.

```plaintext
rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
```
