Create Express App with JavaScript

### 1. Create a New Project Folder

```bash
mkdir my-express-app
cd my-express-app
```

### 3. Install Express

```bash
npm install express
```

---

### 4. Create the Entry File

Create `index.js` inside the project folder:

```js
// index.js
const express = require("express");
const app = express();

// Middleware
app.use(express.json());

// Route
app.get("/", (req, res) => {
  res.send("Hello, Express!");
});

// Start server
const PORT = 3000;
app.listen(PORT, () => {
  console.log(`ðŸš€ Server running at http://localhost:${PORT}`);
});
```

---

### 5. Run the App

```bash
node index.js
```

Visit: [http://localhost:3000](http://localhost:3000)

---
