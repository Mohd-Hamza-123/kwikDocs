**Express.js** is a **backend web framework** for **Node.js** that helps you build web applications and APIs more easily.


### ðŸ”¹ What Express.js is

* A **minimal** and **flexible** framework built on top of **Node.js**.
* It provides a **simpler way** to handle things like:

  * Creating a web server
  * Defining routes (`GET /`, `POST /users`, etc.)
  * Handling requests and responses
  * Using **middlewares** (functions that process requests before sending a response)
* Instead of writing raw Node.js server code, Express gives you a cleaner API.

---

### ðŸ”¹ Why we use Express.js

Without Express, a simple Node.js server looks like this:

```js
const http = require("http");

const server = http.createServer((req, res) => {
  if (req.url === "/") {
    res.write("Hello World");
    res.end();
  }
});

server.listen(3000);
```

With **Express.js**, it becomes:

```ts
import express from "express";

const app = express();

app.get("/", (req, res) => {
  res.send("Hello World");
});

app.listen(3000, () => {
  console.log("Server running at http://localhost:3000");
});
```
