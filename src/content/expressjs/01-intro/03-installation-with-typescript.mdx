### 1) Initialize the project 

```bash
npm init
```

You will be prompted with the following questions:

```
package name: (my-express-app)  my-express-app
version: (1.0.0)  1.0.0
description: A simple Express.js app
entry point: (index.js)  index.js
test command:  (leave blank or type "jest" if you plan to use it)
git repository: (optional, leave blank if not using Git)
keywords: express, node, api
author: Your Name
license: (ISC)  MIT
```

This will create a `package.json` file in your project directory.


### 2) Install Express

```bash
npm install express
```

### 3) Install TypeScript

```bash
npm install -D typescript
```

### 4) Create a `index.ts` File

Create a `index.ts` file in the root directory of your project.

### 5) Installing Type Declarations for Express

```bash
npm install -D @types/express @types/node
```

### 6) Create a `tsconfig.json` File

To create a `tsconfig.json` file, run the following command:

```bash
npx tsc --init
```

This file will look like this:

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "CommonJS",
    "moduleResolution": "Node",
    "rootDir": "src",
    "outDir": "./dist",
    "strict": true,
    "esModuleInterop": true,
    "allowSyntheticDefaultImports": true,
    "verbatimModuleSyntax": false,
    "skipLibCheck": true,
    "types": ["node"],
    "sourceMap": true,
    "declaration": true,
    "declarationMap": true,
    "noUncheckedIndexedAccess": true,
    "exactOptionalPropertyTypes": true,
    "isolatedModules": true,
    "moduleDetection": "force",
    "noUncheckedSideEffectImports": true
  },
  "include": ["src"]
}
```

### 7) Create a `src` Directory

After creating `src` directory , the project structure will look like this:

```
my-express-app/
├── src/
│   ├── index.ts        # entry file
│   ├── routes/         # routes directory (optional)
│   │   └── user.routes.ts
│   └── controllers/    # controllers directory (optional)
│       └── user.controller.ts
├── dist/               # will be created by tsc
├── package.json
├── tsconfig.json
└── .env
```

### 8) Compile the TypeScript Code

Run the following command to compile your TypeScript code:

```bash
npx tsc
```

This will create a `dist` directory with the compiled JavaScript files.

This dist folder will look like this:

```
dist/
├── index.js (entry file)
├── routes/
│   └── user.routes.js
└── controllers/
    └── user.controller.js
```

You can now run your app with `node dist/index.js`.

### 9) Update the `package.json` File

Update the `package.json` file to include the following scripts:

```json
{
  "scripts": {
    "build": "npx tsc",
    "start": "node dist/index.js"
  }
}
```

Now you can run the following command to build your app:

```bash
npm run build
```

And to start your app:  

```bash
npm run start
```

### 10) Install `tsx` 

```bash
npm install -D tsx
```


### 11) Update the `package.json` File

Update the `package.json` file to include the following scripts:

```json
{
  "scripts": {
    "build": "npx tsc",
    "start": "nodemon dist/index.js",
    "dev" : "tsx watch src/index.ts"  // watches & restarts on change
  }
}
```

### 12) Run the App 

```bash
npm run dev
```

Now it will watch for changes and restart the server automatically.
