Following are the ways to dynamically render a static page.

### 1) Using dynamic export

```plaintext
/app
  /services
    page.tsx
```

```tsx
export const dynamic = 'force-dynamic'
export default function ServicesPage() {
    console.log("Running Static Page")
    return <h1>This is services page</h1>
}
```
[Learn more](https://nextjs.org/docs/app/api-reference/file-conventions/route-segment-config#dynamic)

### 2) Using search Params

```tsx
export default async function ServicesPage({searchParams} : any) {
    const search = await searchParams;
    console.log("Running Static Page , dynamically using search Params")
    return <h1>This is services page </h1>
}
```

### 3) Using cookies

```tsx
import { cookies } from "next/headers"

export default async function ServicesPage() {
    const cookie = await cookies()
    console.log(cookie)
    return <h1>This is services page</h1>
}
```

Note : To learn all ways to render static page dynamically visit next js [Documentation](https://nextjs.org/docs/app/building-your-application/rendering/server-components#dynamic-apis)
