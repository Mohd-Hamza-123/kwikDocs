Check username and email address:

```bash
git config --global user.name
git config --global user.email
```

Set username and email address:

```bash
git config --global user.name "Your Name"
git config --global user.email "Your Email"
```

You can also set username and email address for a specific repository:

```bash
git config user.name "Your Name"  # Inside the repository       
git config user.email "Your Email" # Inside the repository
```
