### 🔧 Git Configuration Guide  

Easily check and set your Git username and email to ensure a smooth workflow.  

---

### ✅ **Check Your Current Configuration**  
To view your global Git username and email, run:  

```bash
git config --global user.name  
git config --global user.email  
```  

---

### ✏️ **Set Global Username and Email**  
Set your global Git identity with these commands:  

```bash
git config --global user.name "Your Name"  
git config --global user.email "Your Email"  
```  

- Replace `"Your Name"` with your preferred name.  
- Replace `"Your Email"` with your email address.  

---

### 📁 **Set Username and Email for a Specific Repository**  
To set your Git identity only for a specific repository, navigate to the repository folder and run:  

```bash
git config user.name "Your Name"  
git config user.email "Your Email"  
```  

> 🌟 **Tip**: Using repository-specific configuration is useful when working on projects with different identities.

### 📚 Check your config settings:

```bash
git config --list
```
This command will display all the configuration settings, including your username and email.

### Change default code editor

You can change the default code editor in your system to vscode. To do this, you can use the following command:

```bash
git config --global core.editor "code --wait"
```
